# Tweet - 明確な設計図の力

## 本文

目標が曖昧だと、
行動できぬ。

行動できぬと、
迷走する。

迷走すれば、
時間だけが過ぎていく。

3ヶ月後の理想の未来を明確にし、
そこへの設計図を描くのじゃ。

---

## メタ情報

### 使用した要素
- **Claim**: 理想の未来を明確にし、そこに向かう道筋を設計せよ
- **Proof**: Logos（論理）
- **参照**: 目標が曖昧だと → 行動できない → 迷走する → 時間だけが過ぎる
- **パターン**: 因果関係型
- **スタイル**: 占い師・賢者キャラ
- **文字数**: 99文字

### 構成
1. 負の連鎖を段階的に描写
2. 問題の深刻さ（時間だけが過ぎる）
3. **Claim**（明確にし、設計図を描け）
4. 具体性（3ヶ月後）
