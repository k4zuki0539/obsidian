# Tweet - 強みを見つけた者の変化

## 本文

【相談者からの報告じゃ】

苦手なことで戦い続けていた頃は、
毎日が疲弊との闘いじゃった。

でも、強みを見つけて活かし始めてから、
人生が輝き始めたと。

おぬしの強みは何か？
それを知ることから始めよ。
