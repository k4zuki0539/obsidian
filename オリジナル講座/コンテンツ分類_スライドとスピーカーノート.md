# コンテンツ分類：スライド vs スピーカーノート

## 【基本方針】

### スライドに配置すべき内容
- **視覚的に見せるべき重要なポイント**
- **キーワード・定義・数式**
- **見出し・箇条書き**
- **図表・比較表**
- **簡潔なメッセージ（1スライド1メッセージ）**

### スピーカーノートに配置すべき内容
- **詳細な説明・補足**
- **具体例の詳細・エピソード**
- **話す内容・トーン**
- **時間配分・注意点**
- **聞き手への問いかけ**
- **失敗談・体験談**

## 1. はじめに

### スライド
- タイトル：「ビジネス基礎の重要性」
- メッセージ：
  - 「ビジネスの基礎ができていないと、商品設計や仕組み化の土台ができない」
  - 「需要を見つけることが一番大事」

### スピーカーノート
- 詳細説明：「マネタイズセットの解像度が下がる」という意味を具体例で説明
- 話す内容：「この基礎をしっかり理解することで、その後の商品設計が格段にスムーズになります」
- 時間目安：1分

## 2. ビジネスとは

### スライド1: お金とは何か
**スライド内容：**
- タイトル：「お金とは何か？」
- メッセージ：「お金 = 価値と交換するもの」
- 強調：「お金を稼ぐ = 価値を与えてお金をもらうこと」

**スピーカーノート：**
- 話す内容：「お金を稼ぐことは悪いことではない、というマインドブロックを外すことが重要です」
- 具体例：病院の例、iPhoneの例を詳しく説明
- 注意点：「価値のないものを与えてお金をもらうのは悪いこと」という区別を明確に
- 時間目安：2〜3分

### スライド2: ビジネスの定義
**スライド内容：**
- タイトル：「ビジネスとは？」
- メッセージ：「ビジネス = 需要に対して供給を届ける活動」
- 強調：大きく表示

**スピーカーノート：**
- 話す内容：「これだけ覚えておいてください。すべてのビジネスはこの定義に当てはまります」
- 時間目安：30秒

### スライド3: 商品を買う理由
**スライド内容：**
- タイトル：「なぜ人は商品を買うのか？」
- メッセージ：「現状の状態と理想の状態のギャップを埋めるため」
- 図解：現状 → [ギャップ] → 理想
- 強調：「ギャップが大きいほど、商品の値段は高くなる」

**スピーカーノート：**
- 話す内容：「不満があるだけでは浅い。不満 + 理想があることが重要」
- 具体例：「この差分を埋めるものが商品です」
- 時間目安：2分

## 3. ニーズ・欲求・需要の関係性

### スライド4: ニーズとは
**スライド内容：**
- タイトル：「ニーズとは？」
- 定義：「何かしらの不満や欠乏から必要性を感じている状態」
- 特徴：
  - 「第三者が作り出すことはできない」
  - 「抽象的」
- 例：箇条書き
  - お腹がすいたな
  - 家族と幸せになりたいな
  - 喉が渇いたな

**スピーカーノート：**
- 話す内容：「これだけ聞いても、この人は何が欲しいのかわからない。これがニーズです」
- 時間目安：1分30秒

### スライド5: 人間の8大欲求
**スライド内容：**
- タイトル：「根源的欲求（8大欲求）」
- 8つのリスト：
  1. 人生を楽しみたい
  2. 食べ物を味わいたい
  3. 恐怖から逃れたい
  4. 性的に交わりたい
  5. 快適に暮らしたい
  6. 他人に勝ちたい
  7. 愛する人を守りたい
  8. 社会的に認められたい
- 強調：「これが満たされないと、どうしようもない」

**スピーカーノート：**
- 話す内容：「Life Force、生命力とも言われます。一度感じてしまうと、解消するまで残り続けます」
- 注意点：「ビジネスするときは、必ずこの8大欲求のどこかに当てはまる必要がある」
- 時間目安：2分

### スライド6: 固定的欲求
**スライド内容：**
- タイトル：「固定的欲求（二次的欲求）」
- 特徴：「人間全員が持っているわけではない」
- 例：
  - 情報が欲しい
  - 便利でありたい
  - 節約したい

**スピーカーノート：**
- 話す内容：「その人のライフスタイル、価値観、環境によって出てくる欲求です」
- 時間目安：1分

### スライド7: 欲求とは
**スライド内容：**
- タイトル：「欲求とは？」
- 定義：「ニーズが特定の対象に向けられたもの」
- 特徴：「人によって異なる」「具体的」
- 例（図解）：
  - ニーズ：お腹がすいたな
  - 欲求：お寿司が食べたい / 焼き肉が食べたい / ハンバーガーが食べたい

**スピーカーノート：**
- 話す内容：「ニーズは抽象的、欲求は具体的。ニーズを満たす欲求は複数あります」
- 具体例：家族と幸せになりたい（ニーズ）→ お金を稼いで幸せになりたい / 家族関係を改善して幸せになりたい
- 時間目安：2分

### スライド8: ビジネスとの関係（重要なポイント）
**スライド内容：**
- タイトル：「ビジネスは何に対して供給を届ける？」
- メッセージ：「ビジネス = 需要に対して供給を届ける活動」
- 警告：「欲求に対して供給を届ける活動ではない」
- 強調：「多くの人がやりがちなミス」

**スピーカーノート：**
- 話す内容：「欲求に対して供給を届けてしまうと、うまくいかない。これが稼げない人の典型的な間違いです」
- 具体例：お寿司の例を詳しく説明
- 時間目安：2分30秒

### スライド9: 需要とは
**スライド内容：**
- タイトル：「需要とは？」
- 定義：「欲求をさらに特定の対象に向けた具体的なもの」
- 特徴：
  - 「さらに具体的」
  - 「購買力を伴う願望」
  - 「満たさなくても困らないが、満たされると嬉しい」

**スピーカーノート：**
- 話す内容：「欲求も具体的でしたが、需要はさらに深い、細かい、特定の会社です」
- 具体例：お寿司が食べたい（欲求）→ 高級なお寿司が食べたい / 家族で安く食べられるお寿司が食べたい（需要）
- 時間目安：2分

### スライド10: 需要の見つけ方
**スライド内容：**
- タイトル：「需要の見つけ方」
- 方法：
  1. 欲求を細分化する
  2. ニーズを深掘りする
- 例：ハンバーグの例
  - 欲求：ハンバーグが食べたい
  - 需要：外で食べられる、かつ低価格で、ボリュームのあるハンバーグが食べたい

**スピーカーノート：**
- 話す内容：「ハンバーグをあげても『いや、そのハンバーグいらない』となりかねない。ギャンブルになってしまいます」
- 具体例：マックの例を詳しく説明
- 時間目安：2分30秒

### スライド11: まとめ（ニーズ・欲求・需要）
**スライド内容：**
- タイトル：「ニーズ・欲求・需要の関係性」
- 比較表：
  | | ニーズ | 欲求 | 需要 |
  |---|---|---|---|
  | 特徴 | 抽象的 | 具体的 | さらに具体的 |
  | 作り出せる？ | × | ○ | ○ |
  | 例 | お腹がすいた | お寿司が食べたい | 高級なお寿司が食べたい |

**スピーカーノート：**
- 話す内容：「この3つの違いを理解することが、ビジネス成功の鍵です」
- 時間目安：1分30秒

### スライド12: 良い需要の条件
**スライド内容：**
- タイトル：「良い需要の条件」
- 3つの条件：
  1. 競合が少ないこと
  2. 人数が多いこと
  3. 購買意欲が高いこと
- 各条件の説明（簡潔に）

**スピーカーノート：**
- 話す内容：「この3つを満たす需要を見つけることが重要です」
- 具体例：高級寿司 vs ファミリー寿司の店舗数の違い、がん治療の購買意欲の例
- 時間目安：2分

## 4. 供給とは

### スライド13: ニーズ・欲求・需要の見つけ方（質問方法）
**スライド内容：**
- タイトル：「見つけ方のコツ」
- メッセージ：
  - ニーズ → 「なぜ？」を聞く
  - 欲求・需要 → 「何を？」「どのように？」を聞く

**スピーカーノート：**
- 話す内容：「お金を稼ぎたい（欲求）→ なぜ？→ モテたいから（ニーズ）」
- 具体例：お金を稼ぎたい人の深掘り例を詳しく説明
- 時間目安：2分

### スライド14: 供給ファーストはNG（重要な警告）
**スライド内容：**
- タイトル：「供給ファーストはNG」
- 警告：「需要を見つけずに、自分の売りたいものを先に考えると失敗する」
- 例：ヨガ講師の例
  - NG：「ヨガを教えたい」→ 需要を探す
  - OK：需要を見つける → その需要を満たす方法としてヨガを使う

**スピーカーノート：**
- 話す内容：「ヨガのイメージ価格は安い。5000円ぐらい。誰でも教えられそうな感じ。これでは購買意欲が低い」
- 具体例：「ヨガを使ったダイエットの専門家」というコンセプトの失敗例を詳しく説明
- 注意点：「このマインドがかかっていると稼げない。断言できます」
- 時間目安：3分

### スライド15: 究極の需要から逆算
**スライド内容：**
- タイトル：「究極の需要から逆算する」
- 例：ヨガ講師
  - 需要：「産後3kgまで戻したい、無理なくマイペースに」
  - コンセプト：「産後3kg、無理なく戻すゆったりダイエットの専門家」
  - 供給：ヨガ（表には出さない）
- 比較：
  - NG：「ヨガを使ったダイエットの専門家」（イメージ価格あり）
  - OK：「ゆったりダイエットの専門家」（イメージ価格なし）

**スピーカーノート：**
- 話す内容：「同じヨガを供給していても、コンセプト次第で180度違います」
- 具体例：「お客さんがイメージできないコンセプトにするといい。イメージ価格のないポジションを取る」
- 注意点：「商品があるなら、まず究極の需要を絶対に見つけてください。一旦、自分の売りたいものを頭の中から出してください」
- 時間目安：3分30秒

## 6. 売り上げの計算式

### スライド19: 売り上げの方程式
**スライド内容：**
- タイトル：「売り上げの計算式」
- 数式：「売り上げ = 見込み客数 × 成約率 × 単価 × 購入回数」
- 強調：「すべてのビジネスに当てはまる」

**スピーカーノート：**
- 話す内容：「購入回数はリピートです。この4つから売り上げは構成されています」
- 時間目安：1分

### スライド20: たこ焼き屋の例
**スライド内容：**
- タイトル：「例：たこ焼き屋さん」
- 計算例：
  - 見込み客数：1000人
  - 成約率：10%（100人）
  - 単価：1000円
  - 購入回数：1回
  - 売り上げ：10万円
- リピート例：購入回数2回 → 売り上げ20万円

**スピーカーノート：**
- 話す内容：「どんなビジネスでも、店舗ビジネスでもSNSビジネスでも、同じ計算式です」
- 時間目安：1分30秒

### スライド21: 目標から逆算する
**スライド内容：**
- タイトル：「目標から逆算する」
- メッセージ：「達成可能かどうか、物理的に見極める」
- 例：物販で月100万円
  - 現状：30万円
  - 必要：3.3倍にする
  - 問い：「物理的にできますか？」

**スピーカーノート：**
- 話す内容：「見込み客数を3300人に増やして、毎月継続できますか？成約率を33%に上げられますか？」
- 注意点：「物理的に毎月安定して100万達成できますか？」
- 時間目安：2分30秒

### スライド22: ユニエコとスケーラビリティ
**スライド内容：**
- タイトル：「ユニエコとスケーラビリティ」
- ユニエコ：「1人あたりから得られるお金」
- スケーラビリティ：「継続して拡大していけるか」
- 例：パーソナルトレーニング
  - 1万円 × 3時間
  - 100万円達成には100人必要
  - 300時間必要 → 「できますか？」

**スピーカーノート：**
- 話す内容：「自分の時間をいかに少なくして、多くお金をもらうか。達成できたとしても、スケールできますか？」
- 注意点：「どう考えても高単価で商品を販売した方がいい」
- 時間目安：2分30秒

### スライド23: フロントエンド・バックエンド
**スライド内容：**
- タイトル：「フロントエンド・バックエンド」
- メッセージ：「LTVを1発で回収する」
- 例：
  - フロント：500円のnote
  - バックエンド：80万円のコンサル
- 警告：「フロントエンドを売って終わらせたら詰みます」

**スピーカーノート：**
- 話す内容：「SNSはリピートが難しい。1発で回収しましょう」
- 具体例：noteの例を詳しく説明
- 時間目安：2分

- 時間目安：2分
