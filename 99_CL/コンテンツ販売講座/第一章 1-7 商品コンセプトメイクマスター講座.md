## 1. はじめに - 商品コンセプトとは

### 商品コンセプトの定義

**商品コンセプト = 3つの要素を言語化したもの**

```
どんな悩みを抱えている人を
↓
どのような手段によって
↓
どのような状態にするのか
```

### なぜ商品コンセプトが必要なのか?

**商品コンセプトがないと起こる問題:**
- どういう商品を作ればいいかわからない
- 商品の中身が決まらない
- サポート体制が決まらない
- 商品設計で迷走する

**商品コンセプトがあると:**
- 商品設計の解像度が上がる
- 一貫した商品設計ができる
- 迷わずに進められる

### 商品コンセプトの例

**このコースの商品コンセプト:**

「**SNSビジネス完全未経験、ゼロの方**でも、**独自のコンテンツ販売メソッド**によって、**一生もののコンテンツマーケティングスキルを習得**して、**たった6ヶ月で再現性のある月収30万円以上を実現できる**超実践型オンラインコミュニティ」

## 2. 商品コンセプト設計の最重要ポイント

### 最も重要なこと

```
現状とゴールを具体的に定義する
```

**→ これが商品コンセプト設計で最も重要**

### 現状とゴールの関係

#### 商品の本質

```
現状(見込み客の現在の状態) ──[差]── ゴール(見込み客の理想の状態)
                          ↓
                    この差を埋める手段 = 商品
```

**重要な原則:**
- 現状とゴールの差分を埋めるのが商品
- 現状とゴールが決まっていないと商品設計ができない
- 抽象的な定義では商品設計がうまくいかない

**→ 必ず具体的に定義する**

### 具体例で理解する

#### コースの例

| 要素 | 内容 |
|------|------|
| **現状** | SNSビジネス完全未経験、ゼロの方 |
| **ゴール** | たった6ヶ月で再現性のある月収30万円以上を実現できる |
| **差** | 未経験 → 月収30万円 |
| **手段(商品)** | 独自のコンテンツ販売メソッド |

### 「具体的」とは何か?

#### 定義

**具体的 = 人によって解釈のずれが限りなく少ない状態**
**抽象的 = 人によって解釈のずれが大きい状態**

#### 具体例で比較

| 抽象的な表現 | 具体的な表現 | 理由 |
|------------|------------|------|
| 最速で収益化できる | 6ヶ月で月収30万円の収益化ができる | 期間と金額が明確 |
| 理想の体を手に入れる | 3ヶ月でマイナス10kg、体脂肪率18%を実現 | 期間と数値が明確 |
| 子育てがうまくなる | 3ヶ月で親子関係の満足度が8割以上向上 | 期間と指標が明確 |

**具体的にする方法:**
- 数値を使う(期間、金額、数量など)
- 名詞を使う
- 測定可能な指標を使う

### なぜ具体的に定義する必要があるのか?

**具体的に定義することで:**

```
現状が具体的 × ゴールが具体的 = 商品の解像度が上がる
```

**結果:**
- 商品設計が明確になる
- お客様にも伝わりやすい
- 自分も迷わなくなる

## 3. 商品コンセプト設計の前提

### 大前提: 競合サービスを参考にする

**鉄則:**

```
既に売れている競合サービスの中で、自分にもできそうなサービスを元に考える
```

**理由:**
- 競合分析で「自分でもできそう」と思ったはず
- すでに売れている = 需要がある証拠
- ゼロから考える必要はない

### よくある質問: 差別化は必要?

**答え: 差別化は不要**

**正しいアプローチ:**
- そのまま参考にして大丈夫
- 競合がたくさんいる市場に飛び込む
- 本質は同じでOK

### なぜ差別化が不要なのか?

#### 理由1: 売れるサービスの本質は同じ

**すべての売れているサービスは本質的に同じことを教えている**

**例: ダイエット**

| サービス | 手段 | ゴール |
|---------|------|--------|
| 食事+運動型 | 食事管理と運動 | 理想の体を手に入れる |
| 運動特化型 | 運動だけで痩せる | 理想の体を手に入れる |
| 食事特化型 | 食事制限だけ | 理想の体を手に入れる |

**ポイント:**
- ゴールは同じ
- 手段が少し違うだけ

**例: 子育て**

| サービス | 手段 | ゴール |
|---------|------|--------|
| 褒め方特化 | 褒め方をマスター | 幸せな親子関係 |
| コーチング型 | セルフコーチング習得 | 幸せな親子関係 |
| 食事改善型 | 幼少期の食事改善 | 幸せな親子関係 |

**→ ゴールは同じ、手段が少し違うだけ**

#### 理由2: 革新的なサービスは不要

**革新的なアイデアは必要ない:**
- ネットが普及してから20年以上経っても、まだ使いこなせない人がいる
- 電動キックボードも、電動モーターという既存技術の組み合わせ
- 革新的なサービスで売れているわけではない

**→ すでに売れているものと同じで良い**

#### 理由3: ティッシュの例

**ティッシュの差別化:**
- 本質は変わらない(鼻をかむ、拭く)
- ちょっと柔らかい
- ちょっと香りが良い
- → めちゃくちゃ小さな差別化

**オンラインビジネスも同じ:**
- 本質は一緒
- サポート体制で差別化(チャットサポート、無制限など)
- キャラクター性で差別化
- → 小さな差別化の積み重ね

### 重要な原則

**鉄則:**

```
売れている競合サービスと同じことをする = 売れる
売れている競合サービスと違うことをする = 売れない(革新的すぎる)
```

**理由:**
- 初めて商品設計する人が、売れているサービスを参考にしないで売れる商品を作れるわけがない
- 革新的なサービスを作るのは、20〜40年のキャリアを持つプロマーケターの仕事
- 私たちは既存の成功パターンを活用すべき

### 差別化される要素

**自然に差別化される要素:**

1. **誰から教わったのか**(教える人が違う)
2. **価値観が違う**(考え方、アプローチ)
3. **教え方が違う**(優しい、厳しい、論理的など)
4. **キャラクター性**(性格、話し方)
5. **サポート体制**(充実度、方法)

**→ 同じサービスでも、これらで自然に差別化される**

## 4. 商品コンセプトの5つの構成要素

### 商品コンセプトの構造

**このコースの商品コンセプト(再掲):**

「**①SNSビジネス完全未経験、ゼロの方**でも、**②独自のコンテンツ販売メソッド**によって、**③一生もののコンテンツマーケティングスキルを習得**して、**④たった6ヶ月で再現性のある月収30万円以上を実現できる**⑤超実践型オンラインコミュニティ」

### 5つの構成要素

| 要素 | 説明 | 例 |
|------|------|-----|
| ① 現状 | 見込み客の現在の具体的な状態 | SNSビジネス完全未経験、ゼロの方でも |
| ② 手段 | 自分の成功要因や独自のメソッド | 独自のコンテンツ販売メソッドによって |
| ③ スキル | 手段の先に手に入るスキル | 一生もののコンテンツマーケティングスキルを習得して |
| ④ ゴール | スキル習得後に実現できるゴール | たった6ヶ月で再現性のある月収30万円以上を実現できる |
| ⑤ キーワード | サービスの特徴を表現した言葉 | 超実践型オンラインコミュニティ |

**最も重要:** ①現状と④ゴールが具体的に定義されていること

### 要素1: 現状

**定義:** 見込み客の現在の具体的な状態

**例:**

| ジャンル | 現状の表現 |
|---------|-----------|
| SNSビジネス | SNSビジネス完全未経験、ゼロの方でも |
| 恋愛 | 女性経験ゼロでも |
| ダイエット | 運動が苦手な方でも |
| 美肌 | 何をやってもお肌が改善しなかった方でも |
| 片付け | 片づけが全くできない方でも |

**ポイント:**
- 競合サービスを参考にする
- 具体的に表現する
- 「〜でも」という形で表現することが多い

### 要素2: 手段

**定義:** 自分の成功要因や独自のメソッド

**ポジションメイクで決めた成功要因を入れる**

**例:**

| ジャンル | 手段の表現 |
|---------|-----------|
| 筋トレ | 解剖学から逆算した筋トレフォームによって |
| 子育て | 独自の褒め方メソッドによって |
| コンテンツ販売 | 独自のコンテンツ販売メソッドによって |
| メイク | 魅力を最大限に生かす自分流のメイク術によって |
| 腸活 | 一日3回食べるだけの本質の腸活メソッドによって |
| アンガーマネジメント | アンガーマネジメントによって |

**ポジションが決まっていない人:**
- 「独自の○○メソッドによって」
- 「本質の○○メソッドによって」
- 「○○をベースにした独自メソッドによって」

**→ これで大丈夫**

### 要素3: 身につくスキル

**定義:** 手段の先に手に入るスキル

**フォーマット:** 「〜スキルを習得して」「〜習慣を習得して」

**例:**

| ジャンル | 身につくスキル |
|---------|--------------|
| 美肌・腸活 | 一生ものの美肌腸活習慣を習得して |
| コンテンツ販売 | 一生もののコンテンツマーケティングスキルを習得して |
| 恋愛 | 女性にモテる最強の会話術を習得して |
| ダイエット | 一生続けられる運動習慣を習得して |
| アンガーマネジメント | 一生もののアンガーコントロールスキルを習得して |
| 料理 | 時短する料理スキルを習得して |

**ポイント:**
- 「一生もの」という表現が効果的
- 「習得して」で終わる

### 要素4: ゴール

**定義:** スキル習得後に実現できる具体的なゴール

**フォーマット:** 「たった○○で△△を実現できる」

**例:**

| ジャンル | ゴール |
|---------|--------|
| ダイエット | たった3ヶ月で理想の体型、マイナス10kgを実現できる |
| 美肌 | 最短28日で誰もが振り向く美肌を実現できる |
| 子育て | たった3ヶ月で幸せな親子関係を実現できる |
| コンテンツ販売 | たった6ヶ月で再現性のある月収30万円以上を実現できる |

**ポイント:**
- 期間を明確にする(たった○ヶ月)
- 数値を入れる(具体的にする)
- 「実現できる」で終わる

### 要素5: キーワード

**定義:** サービスの特徴を表現した言葉

**例:**

- 女性限定オンライン講座
- 完全オンライン完結のコーチング
- 短期集中型のインスタグラムマーケティングプログラム
- 完全マンツーマンのコーチング
- 超実践型オンラインコミュニティ

**ポイント:**
- なくても良い(あれば魅力的)
- サービスの特徴を端的に表現
- 「講座」「プログラム」「コーチング」「コミュニティ」などの形態を表現

### 商品コンセプトの具体例

#### 例1: ダイエット

「**運動と食事制限が苦手な方**でも、**一日3回食べるだけの本質の腸活×習慣化メソッド**によって、**一生ものの美肌腸活習慣を習得**して、**たった3ヶ月で体重マイナス10%を実現できる**オンライン講座」

#### 例2: 美肌

「**今まで何をしてもお肌が改善しなかった方**でも、**本質の腸活×習慣化メソッド**によって、**一生ものの美肌腸活習慣を習得**して、**最短28日で誰もが振り向く美肌を実現できる**プログラム」

#### 例3: 子育て

「**子育てでストレスを抱えているお母さん**でも、**アンガーマネジメント**によって、**一生もののアンガーコントロールスキルを習得**して、**たった3ヶ月で幸せな親子関係を実現できる**コーチング」

#### 例4: 片付け

「**片づけが全くできないお母さん**でも、**○○メソッド**によって、**時短する料理スキルを習得**して、**マンツーマン完全オンライン完結**で片づけができるようになる講座」

## 5. よくある質問への回答

### Q: 再現性のある商品を作れる自信がない

**質問:**
「こんなゴールを達成させる自信がない」
「再現性のある商品を作れるスキルがない」
「自分には実績がない」

**→ 多くの人が抱える不安**

### 前提1: 最初から実績はない

**重要な認識:**

```
最初から実績や権威性がある人はいない
```

**例:**
- 最初から自転車に乗れる人はいない
- 最初からプロ野球選手の人はいない
- → 練習や積み重ねで身につける

**実績や権威性は後天的に身につけるもの:**
- 最初からある必要はない
- 後から積み上げていけば良い
- 誰もが後天的に身につけている

**→ 全く問題ない**

### 前提2: プレイヤーと教育者は別

**重要な原則:**

```
プレイヤーとしての優秀さ ≠ 教育者としての優秀さ
```

#### プレイヤーとして優秀な人の弊害

**優秀すぎる人の問題点:**
- 専門用語で話してしまう
- 「なんでこんなことできないの?」と思ってしまう
- 初心者の気持ちがわからない

**→ 教育者として優秀とは限らない**

#### もともとできなかった人の強み

**教育者として優秀になりやすい:**
- 初心者の気持ちがわかる
- 再現性が高い
- 寄り添える
- 丁寧に教えられる

**コンテンツビジネスで求められるもの:**
- 寄り添い
- 丁寧さ
- わかりやすさ
- サポート力

**→ プレイヤーとして優秀である必要はない**

### 前提3: モニター商品として提供する

**解決策:**

```
モニター商品として提供 → 自己検証 → ブラッシュアップ → 正規販売
```

#### モニター商品とは?

**特徴:**
- 初めて作る商品
- 再現性は最初から担保されない(当たり前)
- 安い価格で提供
- 無料でも良い

**前提:**
- 初めて商品を作って再現性が担保されたら、それは天才
- 小学生の時に身長180cm、東京ドームでホームランを打つようなもの
- → 最初から完璧な商品を作れる人はいない

#### モニター商品の提供方法

**期待値を正しく伝える:**

「今回はモニター商品ですので、私自身、まだまだプレイヤーとしても教育者としても未熟な部分があります。お気づきになる点や至らない点が出てくる可能性がありますが、それでもモニター商品として購入していただく形になります。それでも、よろしいですか?」

**サポート体制を充実させる:**
- しっかりサポートする
- 追加講義を提供
- 結果が出るまで無制限サポート

**自己検証を行う:**
- 実際に商品を提供する
- お客様の反応を見る
- 改善点を見つける
- ブラッシュアップする

**→ これで教育者としての優秀さを身につける**

### ちょっと無理するゴール設定の重要性

#### 守りのゴール vs 攻めのゴール

**守りのゴール(NG):**
「自分の実力ならこれができる」
「再現できる範囲内で設計する」

**→ 良い商品にならない可能性が高い**

**攻めのゴール(推奨):**
「ちょっと無理するゴールを設定する」
「自信がないけど、魅力的なゴールを設定する」

**→ 良い商品になる可能性が高い**

#### なぜ攻めのゴール設定が良いのか?

**成長のメカニズム:**

```
高いゴール設定
↓
達成するために試行錯誤
↓
自己検証
↓
成長
↓
結果的に達成できるようになる
```

**例:**
- 月収30万円のゴール vs 月収100万円のゴール
- 月収100万円を目指すからこそ、色々考える
- 試行錯誤して自己検証する
- 結果的に達成できるようになる

**→ ちょっと無理したゴールを設定することで成長する**

#### 責任を持つということ

**重要なマインド:**

```
ちょっと無理するゴール + モニター価格 + 無制限サポート = 成長
```

**具体的な提供方法:**
- モニター価格で提供
- 無料でも良い
- 結果が出るまで無制限サポート
- お客様が満足するまでサポート

**自分への責任:**
- 約束した結果を出す責任
- 自分の実力不足は無責任
- → だからこそ、結果が出るまでサポートする

**成長のサイクル:**
1. ちょっと無理するゴールを設定
2. お客様に提供
3. 試行錯誤・自己検証
4. 自分が成長
5. お客様も満足

**→ これが良いサービスを作る方法**

## 6. コンテンツ販売の本当の価値

### 情報には価値がない時代

**現代の状況:**
- 情報過多の時代
- ChatGPTなどで情報は簡単に手に入る
- ネットで調べればいくらでも出てくる

**→ 情報そのものに価値はない**

### コンテンツ販売の本当の価値

**定義:**

「**情報過多であるこの時代に、キュレーションによって、購入者の情報収集およびラーニングのコストを削減して、最速でお客様の理想を実現する環境を提供すること**」

### 提供する3つの価値

#### 1. 情報収集のコスト削減

**お客様が削減できるもの:**
- 時間(情報を探す時間)
- 労力(情報を選別する労力)
- お金(間違った情報で失敗するコスト)

**提供すること:**
- 必要な情報だけをギュッと凝縮
- 正しい情報を体系的に整理
- 無駄な情報を排除

#### 2. 最速で理想を実現する環境

**環境の提供:**
- 正しい順序で学べる
- 迷わずに進める
- 最短ルートを提供

#### 3. サポート体制で寄り添う

**サポートの価値:**
- 一人で悩まなくて良い
- つまずいた時に助けてもらえる
- モチベーション維持

### ライザップの例

**ライザップの価値:**
- 40万円のサービス
- 「食事制限と運動をするだけ」← これは誰でも知っている
- では、なぜ40万円で売れるのか?

**答え:**
- 情報収集のコストを削減
- 最速で理想を実現する環境
- パーソナルトレーナーのサポート

**→ 情報ではなく、環境とサポートを提供している**

### まとめ

**コンテンツ販売の本質:**

```
情報の提供 ✗
環境とサポートの提供 ○
```

**だからこそ:**
- 最初から完璧な商品でなくて良い
- モニター商品から始める
- サポートで補う
- 自己検証しながら改善していく

**→ これがコンテンツ販売の仕事**

## 7. フレームワークの使い方

### フレームワークの構成

**記入する5つの項目:**

1. **現状:** 見込み客の現在の具体的な状態
2. **手段:** 自分の成功要因や独自のメソッド
3. **身につくスキル:** 手段の先に手に入るスキル
4. **ゴール:** スキル習得後に実現できる具体的なゴール
5. **キーワード:** サービスの特徴を表現した言葉

### 記入のポイント

#### 何も思いつかない人へ

**対処法:**
- 概要欄の具体事例を参考にする
- 競合サービスを見直す
- 複数の候補を出す

#### 複数の候補を出す

**推奨:**
- 1つだけでなく、複数考える
- 違うパターンを試してみる
- 添削でブラッシュアップする

#### マインドマップを活用

**使い方:**
1. 概要欄のマインドマップをコピー
2. 5つの要素を記入
3. 組み合わせて文章にする

### 添削の活用

**添削で行うこと:**
- 現状とゴールが具体的か確認
- 手段とスキルの整合性を確認
- よりわかりやすい表現に修正
- 魅力的なコンセプトにブラッシュアップ

**→ 添削を受けることで、商品コンセプトが洗練される**

## まとめ

### 商品コンセプトの重要ポイント

**1. 商品コンセプトとは**

```
どんな悩みを抱えている人を × どのような手段で × どのような状態にするのか
```

**2. 最も重要なこと**

```
現状とゴールを具体的に定義する
```

- 具体的 = 人によって解釈のずれが限りなく少ない状態
- 数値、期間、名詞を使う

**3. 商品設計の前提**

| 鉄則 | 説明 |
|------|------|
| 競合サービスを参考にする | すでに売れているサービスを元に考える |
| 差別化は不要 | 本質は同じでOK |
| 小さな差別化で良い | サポート体制やキャラクター性で自然に差別化される |

**4. 商品コンセプトの5つの要素**

| 要素 | フォーマット |
|------|------------|
| ① 現状 | 〜でも |
| ② 手段 | 独自の○○メソッドによって |
| ③ スキル | 一生ものの○○スキルを習得して |
| ④ ゴール | たった○ヶ月で△△を実現できる |
| ⑤ キーワード | 完全オンライン完結、など |

**5. よくある不安への回答**

| 不安 | 回答 |
|------|------|
| 実績がない | 最初から実績がある人はいない(後天的に身につける) |
| 再現性が担保できない | モニター商品として提供して自己検証する |
| スキルが足りない | ちょっと無理するゴールを設定して成長する |

**6. コンテンツ販売の本当の価値**

```
情報収集のコスト削減 + 最速で理想を実現する環境 + サポート体制
```

### 成功の方程式

```
ちょっと無理するゴール設定
×
モニター商品での提供
×
無制限サポート
×
自己検証・改善
=
良い商品 + 自分の成長
```

### アクションプラン

1. 競合サービスを見直す
2. 現状とゴールを具体的に定義する
3. 5つの要素を記入する
4. 複数の候補を作る
5. フレームワークに記入して添削提出

**商品コンセプトが決まれば、商品設計の解像度が一気に上がります！**
