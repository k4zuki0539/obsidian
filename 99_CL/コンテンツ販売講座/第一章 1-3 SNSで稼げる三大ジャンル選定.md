## 1. 大ジャンル選定の4つのルール

### ルール1: 市場規模は大きい方がいい

**競合が多い = チャンス**

- 競合が多い = すでに需要が存在している証拠
- 大きい市場ほど参入機会が豊富

#### 差別化について

- 最初から180度差別化する必要はない
- 王道を攻めながら、徐々に積み重ねで差別化していく
- 情報発信、キャラクター性、価値観などで自然に差別化される

#### HARM(ハーム)の法則

**人間の根源的欲求に基づくジャンルを選ぶ:**

- **H:** Health(健康)
- **A:** Ambition(夢、野望)
- **R:** Relation(人間関係、恋愛、家族関係)
- **M:** Money(お金)

**重要ポイント:**
- HARMの法則に基づく市場は人間の根源的欲求なので需要がなくならない
- 人生の永遠の課題 = 需要がなくならない

### ルール2: HARMの法則に基づくこと

上記のHARM(健康・夢・人間関係・お金)のいずれかに該当するジャンルを選ぶ

### ルール3: 成長産業であること

- HARMに基づいていれば基本的にクリア
- 衰退産業は避ける

### ルール4: 経験があること

**自分の経験・好きなこと・得意なことが大事**

#### スイートスポットの公式

```
需要 × 好きなこと = スイートスポット
```

#### 自己分析の重要性

- 自分の強みがわからない場合は、自己分析を実施
- 自己分析フレームを使って経験・体験を深掘り
- 自分ですら気づいていなかった強み、経験、教えられる内容が見えてくる

## 2. よくある悩みと解決策

### よくある2つの悩み

#### 悩み1: 権威性・実績がない

「ジャンルは分かっているけど、権威性や実績が何もないから無理だ」

#### 悩み2: 教えることへの不安

「自分の経験則でうまくいったが、それを教えて再現性があるか不安」

### 解決策1: 権威性は作っていくもの

**権威性は元々あるものではなく、作っていくもの**

- 情報発信で作る
- コンサル実績の積み重ねで作る
- 個別相談の実績で作る
- 誰もがゼロからスタートしている

### 解決策2: 実績がないことが最大の武器

**実績がない人の最大の武器 = 再現性の担保**

#### 実績がある人の弱点

- ハイスペック(高学歴、容姿端麗など)
- 特殊条件があるため、再現性の部分が弱くなる
- 天才の方法は一般人には再現できない

#### 実績がない人の強み

- 特別な才能や条件なしで成果を出した = 再現性が高い
- 「自分でもできそう」と思わせる力
- 「近くのお兄さん・お姉さん」的なポジション

#### 具体例: どちらから学びたい?

**比較1:**
- 天才が3ヶ月前にやっていたゼロイチ達成方法
- **凡人が3ヶ月前にやっていたゼロイチ達成方法** ← こちらが求められる

**比較2:**
- 年商1億円の人が教える、2年前のゼロイチ達成方法
- **月収30万円の人が教える、3ヶ月前のゼロイチ達成方法** ← こちらが求められる

**理由:** 
- より近い状況の人から学びたい
- 再現性が高いと感じる
- 親近感がある

## 3. キュレーションという考え方

### キュレーションとは?

**キュレーション = 情報を収集して、まとめる作業**

- コンテンツ販売市場の商品の8〜9割はキュレーションでできている
- 既存の情報を集めて、体系的にまとめ直したもの

### なぜキュレーションで商品が成り立つのか?

#### 商品の本質

**商品を作る目的 = お客様の需要を満たすこと**

- 商品の中身そのものは、実は何でもいい
- 重要なのは「お客様のゴールを達成できるか」

#### キュレーションの価値

**提供する3つの価値:**

1. **情報の圧縮** - 散在する情報をまとめる
2. **検索時間の削減** - 情報収集の手間を省く
3. **正解への最短ルート** - 何が正しいか分からない状態から、正解を提示

**例:** 
この講座も、様々な情報を20〜30分にまとめることで、検索・学習の工数を大幅に削減している

### キュレーションの実践方法

#### 具体例: プロダクトローンチのカスタマイズ

- 既存の手法(プロダクトローンチ)をベースに
- 自分の経験を加えて
- 特定の場面(インスタストーリー)にカスタマイズ
- = オリジナルコンテンツの完成

#### 最初はキュレーションでOK

**権威性や経験が少ない人の戦略:**

- 自分の経験 + キュレーション = 独自コンテンツ
- 後からブラッシュアップしていけばいい
- 需要を満たすことができれば十分

### 重要なマインド

**キュレーションの本質:**
- お客様の能動的な情報収集の手間を削減する
- 正しい情報を提供して結果を出させる
- ゴールを達成できる再現性があれば、商品の中身は何でもいい

**情報過多の時代だからこそ:**
- 情報を選別・整理してあげることに価値がある
- 最短最速でゴールに導くことが求められる

## 4. 定量分析で答え合わせ

### なぜ定量分析が必要なのか?

**需要がないジャンルでビジネスを進めると、いつまでたっても稼げない**

→ 選んだ大ジャンルが正しいかどうか、客観的に確認する必要がある

### 確認すべき2つのポイント

#### 確認方法1: ライバルがいるかどうか

**ライバルがいない = 正しいジャンルとは言えない**
**ライバルがいる = 需要がある証拠**

**理由:**
- 情報発信者が多い = 市場の需要が大きい
- 需要の大きいところで戦う方が成功しやすい

**確認する判断軸:**
- 各種SNSで大ジャンルを検索した際にライバルがいるか
- Googleで検索した際に競合がいるか
- 書籍がたくさん出ているか
- インフルエンサーがいるか

**→ メインはSNSで確認する**

#### 確認方法2: どれくらい売れているか

**売れている人がいるか、いないか**

**確認する判断軸:**
- SNSの情報発信者が稼いでいるかどうか
- 売上を公開している人がいるか(特にTwitter/X)
- LINE登録者の人数が多いか
- noteランキング上位にいるか
- 書籍の販売ランキングに入っているか

**→ メインはSNSで確認する**

### 定量分析のまとめ

**2つの確認ポイント:**
1. ライバルがたくさんいる → OK
2. 実際に売れている人がいる → OK

**両方クリアしていれば、そのジャンルは参入価値あり**

その後は、王道の中で努力を積み重ねてポジションを取っていく

## 5. 大ジャンル選定のフレームワーク

### フレームワークの4つの質問

大ジャンル選定では、以下の4つの質問に答えていきます:

#### 1. あなたのやりたいことは何か?

- 自分が取り組みたいジャンルは何か
- HARMに当てはまっていれば基本的にOK

#### 2. 見込み客の欲求は何か?

**どのような欲求を満たすのか?**

例:
- インスタで稼ぎたい
- 恋愛で彼氏/彼女を作りたい
- 転職を成功させたい

#### 3. 大ジャンルに競合がいるか?

- ライバルが多数存在するか確認
- 情報発信者が多いか確認

#### 4. 大ジャンルで売れているか?

- 実際に売上を立てている人がいるか確認
- LINE登録者が多い人がいるか確認

### フレームワークの使い方

1. 概要欄にある「大ジャンル選定のフレームワーク」を開く
2. 4つの質問に答えて入力する
3. 入力したフレームワークを添削に提出する
4. 他の人のフレームも参考にする

## まとめ

### 大ジャンル選定の4つのルール

1. **市場規模は大きい方がいい**
   - 競合が多い = 需要がある証拠
   - 差別化は徐々に作っていく
   - HARMの法則に基づくジャンルを選ぶ

2. **HARMの法則に基づくこと**
   - H: Health(健康)
   - A: Ambition(夢、野望)
   - R: Relation(人間関係、恋愛)
   - M: Money(お金)

3. **成長産業であること**
   - HARMに基づいていれば基本的にOK

4. **経験があること**
   - 需要 × 好きなこと = スイートスポット
   - 自己分析で強みを発見

### よくある悩みの解決策

| 悩み | 解決策 |
|------|--------|
| 権威性がない | 作っていくものなので問題なし |
| 再現性が不安 | 実績がないことが武器になる(再現性の担保) |
| 知識がない | キュレーションで商品を作ればOK |

### 定量分析で答え合わせ

✓ **ライバルがいるか** → いればOK
✓ **売れている人がいるか** → いればOK

### アクションプラン

1. HARMに基づいたジャンルを選ぶ
2. 自己分析で強み・経験を洗い出す
3. 定量分析で答え合わせ(ライバル・売上確認)
4. フレームワークに入力して添削提出
