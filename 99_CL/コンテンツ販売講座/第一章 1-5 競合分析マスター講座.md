## 1. はじめに - 競合分析の重要性

### 競合分析とは?

**すでに売れている競合を調べること**

### なぜ競合分析が重要なのか?

**重要な原則:**

```
売れている競合 = 需要がある証拠 = 市場が認めている証拠
```

### ゼロから需要を探すのは間違い

**失敗するパターン:**
- 「こういう需要があるんじゃないか」と想像で商品を作る
- ゼロベースで需要を探そうとする

**なぜ失敗するのか:**
- それは事実ではなく、想像だから
- 需要をゼロから見つけるのは、10〜40年のキャリアを持つプロマーケターの仕事
- SNSビジネスでは、ほぼ不可能

### 正しいアプローチ

**成功する方法:**

```
すでに売れている競合を調べる → 模範解答を見つける → 真似する
```

**重要なマインド:**
- 競合分析 = 自分の模範となる競合を見つけること
- 最初は模範解答を真似する意識でOK
- 成功している人は全員、最初は真似から始めている

### 差別化のタイミング

**真似 → 差別化の流れ:**

1. 最初は模範となる競合を真似する
2. 自分のパーソナリティ・価値観・性格をビジネスに反映
3. 徐々に差別化していく

**→ 最初から差別化しようとしない**

### 自分にできるビジネスを見つける

**判断基準:**

```
競合がすでに成功している × 自分の能力でできる = 参入すべきジャンル
```

**具体例:**
- 恋愛コーチングで売れている人を発見
- 調べてみたが、自分の能力ではできないと判断
- → そのジャンルには参入しない

**重要な視点:**
- すでに売れている人を調べる
- 自分でも同じようなビジネスができるか判断
- ビジネスの方向性が明確になる

**→ これ以外の方法で需要を見つけることは不可能**

## 2. 競合分析のゴール

### 競合分析の2つのゴール

1. **自分の模範となる競合を見つけること**
2. **自分でもできるビジネスを見つけること**

### 成功の方程式

```
すでに売れている商品 × 自分でもできる = 必ず稼げるようになる
```

**重要:** 時間軸の問題で必ず稼げるようになる

---

## 3. どこを見ればいいのか

### 見るべき3つの場所

1. **SNS(個人)** ← メイン(80%)
2. **法人のサービス** (10〜20%)
3. **プラットフォーム** (参考程度)

**→ この3つを見ると、需要のあるサービス・商品が分かる**

### 1. SNS(個人)を見る

#### 優先順位

1. **Instagram** ← 最優先
2. Twitter/X
3. YouTube

#### 売れている人の基準

| 確認項目 | 具体的な指標 |
|---------|------------|
| 受講生数 | ハイライトに受講生が載っている |
| 個別相談の実績 | 個別相談の人数が多い |
| LINE登録者数 | LINE公式の登録者が多い |
| 受講生の声 | 実績やレビューが多数ある |

**注意:** 売上実績はあまり公開されないので、上記の指標で判断する

### 2. 法人を見る

#### 売れている法人の基準

- Google検索で上位に表示される
- 広告費を多く使っている
- 業界ランキングで上位にいる

#### 法人を見るメリット

**法人は商品力に優れている:**
- 商品の内容・構成が優れている
- プランの作り方が参考になる
- 商品設計のヒントが得られる

### 3. プラットフォームを見る

#### 主なプラットフォーム

- Brain
- note
- Udemy

#### 売れている人の基準

- ランキング上位にいる
- 販売数が多い

**注意:** あれば参考にする程度でOK(なくても問題なし)

### 競合分析の数

#### 推奨する分析数

| カテゴリ | 推奨数 | 備考 |
|---------|--------|------|
| SNS(個人) | **10〜15人** | メイン。必ず調べる |
| 法人 | 1〜10社 | 参考程度 |
| プラットフォーム | あればでOK | なくても問題なし |
| **合計** | **約30人** | 可能であれば |

**実績がある人の場合:** 模範解答となる競合を5人見つければOK

### なぜたくさん分析するのか?

**重要な原則:**

```
分析の量 ∝ ビジネスの解像度
```

**分析すればするほど:**
- ビジネスの解像度が上がる
- 自分の方向性が明確になる
- 商品の作り方・売り方が理解できる

**結果:**
- かなりの確率で、時間の問題で稼げるようになる

## 4. 何を見るのか(5つの要素)

### 競合分析の5つの要素

1. **誰にやっているか**(ターゲット)
2. **何を売っているか**(商品・サービス) ← **最重要**
3. **どこで売っているか**(販売場所)
4. **いくらで売っているか**(価格) ← **重要**
5. **どのように売っているか**(マーケティング)

**→ これが競合分析の原理原則**

### 重要度の優先順位

| 要素 | 重要度 | 理由 |
|------|--------|------|
| 何を売っているか | ★★★★★ | 商品が最も重要 |
| いくらで売っているか | ★★★★☆ | 価格設定の参考になる |
| どのように売っているか | ★★★☆☆ | マーケティングの参考 |
| 誰にやっているか | ★★☆☆☆ | ターゲット理解 |
| どこで売っているか | ★☆☆☆☆ | SNSならあまり重要でない |

**特に重要:** 「何を売っているか」と「いくらで売っているか」の2つ

---

## 5. 各要素の詳細解説

### 要素1: 誰にやっているか(ターゲット)

#### 見るべき内容

**具体的なターゲットを見る:**
- 30代女性などの抽象的な表現はNG
- 悩みの部分・需要の部分を具体的に把握

#### なぜ具体的なターゲットが重要か?

**重要な原則:**

```
競合のターゲット = 事実に基づくペルソナ = 失敗しない
自分で想像したペルソナ = 事実に基づかない = 失敗する
```

#### 具体例: パーソナルジムのターゲット分析

| ブランド | キャッチコピー | ターゲット |
|---------|--------------|-----------|
| ライザップ | 結果にコミット | 短期間で確実に理想の体を目指したい人 |
| フィールサイクル | 努力しなくても楽しそうに | 努力せず楽しく理想の体を目指したい人 |
| 24/7ワークアウト | 2ヶ月で痩せる、分割払いで4000円 | 短期間で低単価で理想の体を目指したい人 |

**ポイント:** 似ている商品でも、切り口で差別化している

### 要素2: 何を売っているか(商品・サービス)

**→ これが最も重要な要素**

#### 重要なマインド

**大原則:**

```
競合と同じ商品を売る = 売れる
```

**「パクリになるのでは?」という疑問への回答:**

**ティッシュの例:**
- 世の中に多数のティッシュメーカーが存在
- ティッシュの違い: 柔らかさ、香り、品質など
- → めちゃくちゃ小さなところで差別化している

**SNSビジネスも同じ:**
- キャラクター、パーソナリティ、アイデンティティで差別化
- 小さな差別化を積み重ねる = 大きな差別化になる
- → 全く問題ない

**結論:** 売れる商品を参考にして同じような商品を作る方が、売れる確率が高く、楽

#### キュレーションの重要性

**競合を10人調べる効果:**
1. 10人の商品を調べる
2. 自分なりにまとめる
3. 自分の経験や工夫を加える
4. = 確実にオリジナルになる

#### お客様の成果を出すために

**商品を作る目的:**

```
商品 = 作りたいものを作る ✗
商品 = お客様の成果を出すために作る ○
```

**重要な警告:**

```
競合の商品と違う商品を作る = 成果を出せる確率が下がる = 詐欺商品になる
```

**正しいアプローチ:**
- 競合の商品を模範解答として参考にする
- キュレーションで作る
- 真似して作る
- → 成果が出る商品になる

#### 見るべき具体的な内容

1. **顧客をどのような状態にするのか**(ゴール)
2. **目次・ロードマップ**
3. **カリキュラムの内容**
4. **提供方法**(オンライン、動画、テキストなど)
5. **サポート体制**(個別、グループ、コミュニティなど)
6. **限定特典**

### 要素3: どこで売っているか(販売場所)

**SNSビジネスの場合:** あまり重要ではない

**見るべき内容:**
- 使用しているツール
- 販売LP
- 決済システム

### 要素4: いくらで売っているか(価格)

**→ 2番目に重要な要素**

**見るべき内容:**
- プランごとの価格
- 分割回数と値段
- 決済手数料
- 決済システム
- カウンセリング料金の有無

### 要素5: どのように売っているか(マーケティング)

#### セールスファネル(販売動線)を見る

**3つのステップを確認:**

1. **集客:** どこでどのように集客しているか
2. **教育:** どこでどのように教育しているか
3. **販売:** どのように販売しているか

**→ この一連の流れを必ず調べる**

## 6. 競合分析フレームワーク

### フレームワークの使い方

1. 概要欄にあるシートをダウンロード
2. 競合をA、B、C、D、E...と分けて分析
3. 5つの要素を記入していく

### 分析のポイント

**見るべき要素の優先順位:**
1. 何を売っているか(商品) ← **最重要・詳しく見る**
2. いくらで売っているか(価格)
3. どのように売っているか(マーケティング)
4. 誰にやっているか(ターゲット)

### 記入のコツ

- 自分が後で見返した時に分かりやすく書く
- リンクがあれば貼っておく
- 講師の事例を参考にする

### 添削基準

**ビジネスの方向性が理解できているかを確認:**
- 自分の模範となる競合を見つけられているか
- ビジネスの方向性を理解できているか
- 書き出しが薄すぎないか

**→ これができていないと、次に進めない**
