## 1. はじめに - ポジションメイクとは

### ポジションメイクの定義

**ポジションメイク = SNSにおける自分の立ち位置を決めること**

### ポジショニングのゴール

```
自分が何者なのかを明確にする
↓
初めての人も一瞬で理解できるようにする
↓
自分の魅力を感じてもらう
```

---

## 2. ポジションメイクの全体像(2ステップ)

### ポジションメイクの2ステップ

**ステップ1:** 自分のジャンルにおける立ち位置を明確にする
**ステップ2:** その立ち位置が伝わる魅力的な名前をつける

**→ この2ステップでポジションメイクは完了**

---

## 3. ステップ1: 立ち位置を明確にする

### 立ち位置を決める極意

**重要な問い:**

```
自分が語る資格を持つようになるまで、何をやって成功できたのか?
```

**言い換えると:**
- 今の自分があるのは、何をしたからか?
- どうやって今の自分を作り上げたのか?
- 自分の成功した要因は何か?

**→ 自分の成功要因 = 自分の最大の強み**

### 成功要因の見つけ方

#### ステップ1: ビフォーアフターを確認

**自分が乗り越えた深刻な悩みを思い出す:**
- 子育てに悩んでいた
- 筋トレ・ダイエットに悩んでいた
- 恋愛に悩んでいた

#### ステップ2: 成功要因を特定

**問いかけ:**
- その悩みを乗り越えて成功した際、何が要因だったのか?
- 最も自分の成功に導いてくれた手段・要因は何か?
- 自分が一番頑張ったことは何か?

**→ これが自分の最大の強み**

### 具体例1: 筋トレ

#### ゴール設定
- 筋肉をつけてマッチョになりたい
- 理想の体を手に入れたい

#### 様々な解決策
| 解決策 | 内容 |
|--------|------|
| ジムに通う | トレーニング環境を整える |
| 正しいフォームを学ぶ | 解剖学に基づいた筋トレ |
| 食事管理 | タンパク質摂取、バランス |
| ランニング | 有酸素運動で脂肪燃焼 |

#### 自分の成功要因を特定

**問い:**
- マッチョになった要因・秘訣は何だったか?
- マッチョになるために何をやっていたか?

**例の答え:**
- **解剖学を理解して、正しい筋トレのフォームを行ったこと**

**→ これが自分の最大の強み = 立ち位置**

### 具体例2: 子育て

#### ビフォーアフター
- Before: 子育てが苦手だった
- After: 子育てが得意になった

#### 成功要因の特定

**問い:** 子育てが得意になった最大の成功要因は?

**例の答え:**
- **褒め方をマスターしたこと**

**→ 自分は「子育てジャンルの褒め方の専門家」になれる**

### ポジショントークができるようになる

#### ポジショントークとは?

**数ある解決策の中でも、自分のメソッド・手段が最も素晴らしいと主張すること**

#### ポジショントークの例(筋トレ)

**主張:**
「マッチョになりたいなら、正しいフォームを身につけて筋トレすることができれば、それさえできればマッチョになれる」

**他の選択肢を間接的に否定:**
「食事制限やアプリに時間をかけるよりも、とにかくフォームをマスターすることが重要」

**効果:**
- ポジションの信頼性が生まれる
- 信ぴょう性が高まる
- 正当性が伝わる
- 論理的な情報発信ができる

**→ ポジショントークができると、めちゃくちゃ売れる**

### ステップ1のまとめ

**やるべきこと:**

1. 自分がビフォーアフターを乗り越えた成功要因・強みを書き出す
2. そのジャンルにおける自分の立ち位置を考える
3. 立ち位置を言語化する

**言語化の例:**
- 「マッチョになりたいなら、解剖学を理解すればいい」
- 「子育てうまくいかないなら、褒め方をマスターすればいい」

## 4. ステップ2: 魅力的な名前をつける

### なぜ名前が必要なのか?

**立ち位置を明確にしただけでは不十分:**
- そのままでは魅力的ではない
- ユーザーにフォローしてもらいにくい

**→ より魅力的な名前に変換する必要がある**

### 名前の付け方の例

#### 例1: 筋トレ

**立ち位置:**
「マッチョになりたいなら、解剖学を理解して筋トレすればいい」

**魅力的な名前:**
- 「本質の筋トレへ導く」
- 「解剖学ベースの筋トレコーチ」

#### 例2: 子育て

**立ち位置:**
「子育てうまくいかないなら、褒め方をマスターすればいい」

**魅力的な名前:**
- 「褒め方で幸せ子育て」
- 「褒め方の専門家」
- 「幸せ子育て 褒め方マスター」

## 5. 名前付けの5つのポイント

### ポイント1: 端的に表現する

**原則:** 無駄な言葉を使わない

**NG例:**
- 「インスタグラムのファン化集客専門家」← 長すぎる
- 「売上アップ支援のコンサルタント」← 長すぎる

**重要:**
- 一瞬で理解できること
- 無駄な部分を削ぎ落とす
- 必要な部分だけを端的に表現

### ポイント2: 名前の印象を大切にする

**ユーザーが初めて見た時の印象が重要**

**良い印象を与える名前:**
- この人、魅力的だな
- 私が求めているものがありそう
- 面白そうだな
- 自分が欲しい理想のゴールがありそう

**NG例:**
- 「親の幸せに寄り添う」← つまらない印象
- 「〜から脱却させる」← 暗い印象

### ポイント3: 陳腐化した表現は使わない

**避けるべき表現:**
- 「最強の〜」
- 「最速の〜」

**理由:**
- 陳腐化している
- ありふれている
- 他の言葉で代用できるなら避けるべき

### ポイント4: 意味がわからない名前にしない

**避けるべきこと:**
- 専門的すぎる謎の言葉
- 誰も知らない造語
- 理解できない表現

**→ 誰でも理解できる名前にする**

### ポイント5: 検索されやすい名前にする

**原則:** ユーザーが検索するキーワードを入れる

**例: インスタ関連のビジネス**

| 検索されやすい | 検索されにくい |
|--------------|--------------|
| インスタ収益化 | インスタコンテンツ販売 |

**理由:**
- 「インスタ収益化」に興味がある人の方が多い
- 検索している人が多い = 集客しやすい

**→ 検索されやすいキーワードを意識する**

## 6. ネーミングの極意

### ネーミングの本質

**極意:**

```
自分の伝えたいイメージ = 相手が感じるイメージ
```

**→ この2つを一致させることが重要**

### 具体例で理解する

#### 伝えたいイメージ
「解剖学を学べばマッチョになれる」
「解剖学を理解して正しいフォームを身につけてほしい」

#### 相手が感じるイメージ
名前を見た時に感じる印象

#### 良い名前の条件

**イメージが一致している名前:**
- 伝えたい価値観が伝わる
- ポジションが明確に伝わる
- ファン化しやすい

### ネーミングのコツ

#### 1. 類義語を調べる

**例: 「最速」の類義語**
- 早い
- スピード
- 迅速
- 即効
- 短期間

**→ より魅力的な表現を見つける**

#### 2. SNSで他の発信者をリサーチ

**チェックポイント:**
- どんなネーミングを使っているか
- どんな表現が魅力的か
- どんなキーワードが使われているか

#### 3. 複数の候補を出す

**フォーマット:**

```
[ジャンルの悩み]を解決したいなら、[これ]ができさえすればいい
```

**例:**
- 「マッチョになりたいなら、本質の筋トレで導く」
- 「子育てで悩むなら、褒め方をマスターすればいい」

**→ 複数考えて、最も魅力的なものを選ぶ**

## 7. 何もない人はどうするか

### 前提: ビフォーアフターがない人

**該当する人:**
- 悩みを解消したことがない
- コンプレックスを乗り越えた経験がない
- ビフォーアフターがまだない

**結論:** 問題ない、方法はある

### 解決策: 情報発信しながら悩みを克服する

**戦略:**

```
情報発信 + 悩みの克服 = 同時進行
```

**手順:**
1. とりあえず情報発信を始める
2. 同時に、悩みの克服に向けて頑張る
3. ビフォーアフターを作りながら発信する

**重要な認識:**
- ビフォーアフターがないと、すぐには稼げない
- 時間がかかる
- ビフォーアフターを超える時間が必要

**→ 時間を短縮するために、同時進行で頑張る**

### 具体例: 恋愛ナンパ

**状況:**
- 恋愛経験が少ない
- ナンパのスキルがない

**戦略:**
- 3ヶ月かけてナンパの実践
- 同時に情報発信を行う
- 学んだことをアウトプットしながら発信

**効果:**
- 学習効率が上がる
- 実績を作りながら発信できる

### とりあえずポジションを作る方法

**万人受けポジションで始める:**

**例:**
- 「インスタ収益化」
- 「インスタ初心者向け」
- 「わかりやすいインスタ運用」

**NG例:**
- 「インスタ収益化のプロ」← プロと名乗らない
- 「インスタ収益化の専門家」← 専門家と名乗らない

**→ まずは基本的なポジションで始める**

### 何もない人用の戦略

**ステップ:**

1. **万人受けポジションを作る**
   - 基本的なポジション
   - 自分でもできそうなポジション

2. **情報発信を開始**
   - とりあえず発信を始める

3. **悩みの克服に取り組む**
   - 同時進行で頑張る

4. **ビフォーアフターができたら**
   - プロパーなポジションに変更
   - より専門的なポジションにシフト

## 8. フレームワークの使い方

### ビフォーアフターがある人用

#### 記入項目

1. **自分がビフォーアフターを乗り越えた成功要因は何か?**
   - 複数あれば全部書き出す
   - その中で一番重要なものにマークをつける

2. **あなたのジャンルにおける立ち位置はどこか?**
   - 成功要因をもとに立ち位置を考える
   - 簡潔に書く

3. **[悩み]を解決したいなら、[これ]ができさえすればいい**
   - 立ち位置を言語化する
   - 自分の成功要因との関係を明確にする

#### ステップ2: 魅力的な名前

**複数の候補を出す:**
- 3つだけでなく、たくさん考える
- 端的に表現する
- 5つのポイントを意識する

**例:**
- 「幸せ子育て 褒め方専門家」
- 「褒め方で幸せに子育て」
- 「子育て 褒め方マスター」

### 何もない人用フレームワーク

#### 記入項目

1. **売れている人のポジションをリサーチ**
   - SNSで売れている人を10〜15人調べる
   - それぞれのポジションを書き出す

2. **自分でもできそうなポジションを見つける**
   - 「これなら自分にも価値提供できそう」
   - 「これなら自分でもできそう」
   - 「この切り口は面白そう」

3. **リストアップする**
   - 良いと思ったものをピックアップ
   - 1〜2つに絞り込む

4. **ポジションを決めて発信開始**
   - とりあえず始めてみる
   - やりながら調整していく

**重要なポイント:**
- 「自分でもできそう」と感じるものは、本当に自分に合っている可能性が高い
- 自分の価値観に合っているポジションを選ぶ
