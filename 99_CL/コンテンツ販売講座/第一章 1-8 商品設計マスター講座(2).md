## 2. 保証の設計

### 保証の重要性

**効果:**
- 成約率が向上する
- 顧客が成果を出す確率が上がる

**→ 絶対に入れるべき**

### おすすめの保証

**推奨する保証:**

| 保証の種類 | 内容 |
|-----------|------|
| 講座代金回収保証 | 講座代金分を稼げるまでサポート |
| ゼロイチ達成保証 | ゼロイチを達成するまでサポート |
| 永久サポート保証 | ゴール達成まで永久にサポート |

**→ アフターサポート系の保証が最適**

### 返金保証は入れない方がいい

#### 返金保証のデメリット

**問題点:**

1. **お客さんが全然やらない事態に陥る**
   - 返金保証があるから、本気でやらない
   - 「返金してもらえばいいや」となる

2. **自分が腹立つ**
   - 命懸けで作った商品
   - サポートもちゃんとやった
   - → それで返金されたら腹立つ

3. **長期的に自分が稼げなくなる**
   - 結果が出ない → 実績がたまらない
   - 実績がない → 稼げない

**→ 返金保証は入れない**

### アフターサポート保証は絶対に入れる

#### アフターサポート保証とは?

**定義:**

```
結果が出るまで保証する = ゴールを約束する
```

**具体例:**

| ジャンル | アフターサポート保証の内容 |
|---------|------------------------|
| 稼ぐ系 | 6ヶ月で月収10万円稼げなかったら、稼げるまでサポート |
| 美肌 | 3ヶ月でニキビを全部なくす。なくならなかったら、なくなるまでサポート |
| TOEIC | TOEIC2級受かるまでサポート |
| 転職 | 大手企業に転職できるまでアフターサポート |

**→ 非稼ぐ系でも全く関係なく適用できる**

#### 重要な原則

```
結果が出るまでサポートする = ゴールを約束する = 最強の保証
```

**→ 絶対に入れてください!(2度目の強調)**

### アフターサポート保証の意外なメリット

#### メリット1: 質問が来ない

**驚きの事実:**
- 永久保証をつけると、質問が来なくなる

**理由:**
- コンサルすると、リテラシーが高くなる
- 頭が良くなる
- 能力がつくと、疲弊するような質問が来ない

**ダメな質問の例:**
- 「スプレッドシートの閲覧権限の解放の仕方、わかりません」← Googleで調べれば出てくる
- 「YouTubeの限定公開のやり方がわかりません」← Googleで出てくる

**→ リテラシーが高くなると、こういう質問が来なくなる**

#### メリット2: 価値観が同じ人になる

**効果:**
- 6ヶ月教えると、価値観が同じになる
- リテラシーが高くなる
- 軽い質問しか来ない

**時給換算の考え方:**
- 一日5分減る × 365日 = 1,825分/年
- でも、将来的な売上(紹介・人脈)で100万円増える
- → 圧倒的にメリットの方が上回る

#### メリット3: 対談動画の撮影ができる

- その人とずっと人脈を持てる
- 対談動画が作れる
- 実績として使える

#### メリット4: 講師・社員への引き上げ

**最大のメリット:**

```
自分の価値観・マインドを刷り込んだ人 = 最高の仲間
```

**理由:**
- マインドが同じ人が社員になると、めちゃくちゃいい
- 仕事が楽しい
- 結果が出るのが早い
- 事業がめちゃめちゃスケールする

**例:**
- 週7で一日10時間作業するマインド
- VS 一日7〜8時間しか働かないマインド
- → どちらが仕事しやすいか?

**実際の事例:**
- サロンのメンバーが全員社員になっている事例
- 講座生が講師に引き上げられる事例

**→ 絶対に関係を持っていた方がいい**

## 3. 限定特典の設計

### 限定特典の重要性

**効果:**
- 成約率がめちゃくちゃ上がる

**→ 絶対に入れる**

### 限定特典の3つのポイント

| ポイント | 説明 |
|---------|------|
| 1. メインコンテンツの弱点を補うもの | メインで足りない部分を特典で補う |
| 2. メインに負けないクオリティのもの | 特典を舐めない、豪華にする |
| 3. 特典単体でも商品を購入したいと思ってもらえるくらいのクオリティ | 特典だけで価値がある |

### ポイント1: メインコンテンツの弱点を補うもの

**考え方:**
- あらかじめ突っ込まれそうなところを特典に入れておく

**例: SNSマーケティング講座の場合**

| 想定される質問 | 補完する特典 |
|-------------|------------|
| デザインを学べないんですか? | キャンバデザインマスター講座を特典に |
| インスタ運用自体は学べないんですか? | インスタ運用講座を特典に |

**→ メインの弱点をあらかじめ補っておく**

### ポイント2: メインに負けないクオリティ

#### 特典を舐めるな!

**ダメな特典の例:**
- シート1枚だけ
- めちゃくちゃしょぼい特典
- メインコンテンツがあるから、特典はいいやと妥協

**→ 特典を舐めてる!**

#### 正しい特典の作り方

**重要な考え方:**

```
この特典があるから欲しい
この特典があるから、今日買わなきゃダメ
```

**推奨する特典:**

| 特典の種類 | 例 |
|-----------|-----|
| 定性的な価値 | お茶会権利、オフ会の参加権利、無料参加権利 |
| 定量的な価値 | テンプレート、フレームワーク、ツール |

**→ とにかく特典を舐めるな!**

### ポイント3: 特典単体で商品の単価が回収できるクオリティ

**基準:**

```
30万円の商品 → 30万円分の価値を特典で感じてもらう
```

**→ めちゃくちゃ豪華にする**

### 特典設計のおすすめ

#### 工数がかからない特典

**推奨する特典:**

| 特典の種類 | メリット |
|-----------|---------|
| テンプレート系 | 一度作れば使い回せる |
| 講座のアップグレード保証 | 講座がアップグレードされるのが見れる |
| 専用サイト見放題 | コストかからないけど豪華に見える |

**例: 専用サイト見放題**
- 本来6ヶ月で見れなくなる設定
- でも、購入者には見放題で提供
- → コストかからないけど、豪華な特典に見える

### 特典設計で重要なこと

**原則:**

```
喉から手が出るほど欲しい特典を作る
```

**方法:**
1. **競合をめちゃめちゃ見る**
2. **競合のセミナー・個別コンサルに参加する**
3. **画面のスクショを撮る**(流出ダメ)
4. **何を上げているのか分析する**

**→ めちゃくちゃ勉強になる**

## 4. 分割決済の設計

### 分割決済の重要性

**効果:**

```
成約率が圧倒的に向上する
```

**→ 絶対に作った方がいい**

### おすすめの分割回数

**推奨:**

| 商品価格 | 推奨分割回数 |
|---------|------------|
| 30万円未満 | 12分割 |
| 30万円以上 | 24分割 |

**注意点:**
- 手数料込みで設定
- 分割手数料を乗っけた上での24分割にする

### 決済会社

#### 推奨する2つの決済会社

| 決済会社 | 特徴 | 審査 |
|---------|------|------|
| Stripe | 簡単に導入できる | 比較的易しい |
| Expa(エキスパ) | 24分割でも回収できる | やや難しい |

**ゼロイチの場合:**
- 審査が難しい可能性がある
- LPが必要だったりする

#### StripeとExpaの違い

**Stripeの問題点:**
- 24分割を入れると、クレジットカードの期限が切れた場合に回収できない
- Stripeがバンされると回収できない

**Expaの強み:**
- 決済額が30万円なくても、30万円の商品を24分割で申し込める
- 回収率が高い

**重要な考え方:**

```
いい商品を提供 × 永久サポート = 24分割入れても回収できる
```

### infotop(インフォトップ)という選択肢

**特徴:**
- 30万円のクレジットカード枠があれば、24分割がつけられる
- 決済額が30万円なくても、申し込める

**→ 検討の価値あり**(詳細は別講義で解説予定)

### まとめ

**重要ポイント:**
- 分割を入れた方がいい
- 詳細は別講義で解説予定

**→ とりあえず「分割した方がいい」と覚えておく**

## 5. サポート体制の設計

### サポート体制の重要性

**ここがめちゃくちゃ大事!**

### サポート体制設計のメリット

| メリット | 説明 |
|---------|------|
| 成約率が上がる | 安心材料になる |
| 成果を出す確率が上がる | サポート充実で結果が出る |
| 口コミが発生する | 結果が出ると自然に広がる |
| 顧客獲得単価(CPA)が下がる | SNS上のお客様の声で成約率UP |

**→ サポートをめちゃめちゃ充実させる**

### サポートを舐めてる人

**よくある間違い:**

```
チャットサポートしかない = サポートを舐めてる
```

**重要な認識:**

```
チャットサポート = 当たり前(今は標準装備)
```

**→ チャットサポートは当たり前、それ以上を提供する**

### サポート体制に入れるべきもの

#### 必須のサポート

| サポートの種類 | 説明 | 対象 |
|-------------|------|------|
| チャットサポート | 当たり前 | 全員 |
| 講義動画形式フィードバック | Loom使用(絶対入れる) | 全員 |
| 個別添削 | 当たり前 | 全員 |
| タスク管理 | 次に何をすればいいか明確にする | 全員 |

#### 講座形式の場合に追加すべきもの

| サポートの種類 | 説明 |
|-------------|------|
| 月1〜2回のZoomコンサル | 個別またはグループ |
| 月1〜2回のグループコンサル | 講座形式に最適 |
| 録画配布 | 講義の録画を配布 |

**→ 講座形式の人の方が、サポート体制を充実させる必要がある**

### 講義動画形式フィードバック(Loom)

**Loomを絶対に入れる理由:**

| 理由 | 説明 |
|------|------|
| 自分も楽 | タイピングしなくて済む |
| お客さんもわかりやすい | 動画で見れる |
| 頭を使わなくて済む | 話すだけでOK |

**チャットで送られてきたら:**
- 長すぎて意味不明
- タイピング打ちたくない
- めんどくさい

**→ 絶対に動画形式フィードバックを入れる**

### サポート体制設計の考え方

**3つの観点:**

1. **顧客の成果により早くつながるサポート体制**
2. **顧客があったら嬉しいなと思うようなサポート**
3. **競合のデメリットを補強するようなサポート**

**例:**
- 競合が抽象度が高すぎる → 超分かりやすいフレームワークを提供
- 競合がサポートが薄い → めちゃくちゃ充実したサポート

### 競合分析から始まる

**重要な原則:**

```
サポート体制の設計 = 競合分析から始まる
```

**やること:**
1. 競合がどういうサポート体制をやっているか分析
2. すでに結果を出している講座のサポート体制を知る
3. 売れている競合のサポート体制をバンバンパクる

**→ めちゃめちゃ競合を分析する**

### サポートコストを減らす工夫

#### なぜサポートコストを減らすのか?

**ビジネス的な観点:**

```
サポート回数を減らす × 結果を出させる = 自分の時間が増える
```

**自分の時間が増えると:**
- 新商品を作れる
- 集客に時間を使える
- セールスファネルを改善できる
- マーケティングを勉強できる
- → ビジネスがどんどんスケールする

**逆にサポートばかりしていると:**
- ビジネスがスケールできない

**→ ビジネスをスケールさせるために、サポート数をいかに減らして結果を出せるかを考える**

#### サポートコストを減らす方法

**1. フレームワーク・ロードマップを用意**

| 用意するもの | 効果 |
|-----------|------|
| フレームワーク | 自発的に行動できる |
| ロードマップ | 「次何やればいいですか?」が減る |
| テンプレート | タスク管理の手間が省ける |

**→ サポートコストが大幅に減る**

**実績:**
- 一日1〜1.5時間程度のサポート時間
- 10人〜30人見てもこの程度で抑えられる

**2. チャット対応を減らす工夫**

| 方法 | メリット |
|------|---------|
| ボイスメッセージ | タイピングしなくて済む |
| 動画形式フィードバック(Loom) | めちゃくちゃ楽 |
| 文章で返さない | 時間短縮 |

**→ サポートコストを減らしながら、お客さんの成果を出させる**

### 質問ルールの重要性(余談)

**ビジネスマンとして重要:**

```
すごい人たちとの関わり方 = 自分がスケールするかどうかが決まる
```

**重要な認識:**
- すごいインフルエンサーとつながった
- 意味わからない質問を発した時点で終わり
- → すごい人は、そういう目がすごい

**気遣い・気配りができる人:**
- インフルエンサーから好かれる
- 起業家から好かれる
- いろんな仕事をもらえる
- 関わってもらえる

**→ こういうルールを作っている理由**

### まとめ

**重要ポイント:**
- 競合をちゃんと見る
- 競合がどういうサポート体制をやっているか見る
- いいとこを真似する

## 6. ステップ5: 価格設定

### 価格設定の重要性

**価格がめちゃくちゃ大事!**

```
価格設定 = 商品の成否を分ける
```

**よくある間違い:**
- オンラインサロンで3,000円でやろうかな
- noteの買い切り低単価商品
- → 絶対にダメ!

### 高単価商品一択

**結論:**

```
高単価商品一択
```

**絶対に売ってはいけない商品:**
- オンラインサロン
- noteの買い切り低単価商品

**絶対にやるべき商品:**
- 高単価商品
- マンツーマンコンサル
- 買い切り型の講座

**→ 絶対に高単価にする**

### 高単価商品のメリット

#### メリット1: サポート人数が少なくて済む

**具体例で比較:**

| プラン | 単価 | 人数 | 売上 | サポート人数 |
|--------|------|------|------|------------|
| プランA | 20万円 | 5人 | 100万円 | 5人 |
| プランB | 50万円 | 2人 | 100万円 | 2人 |

**プランBの圧倒的なメリット:**
- 3人分のサポート人数が減る
- 3人分のリソースが浮く
- この2人にめちゃくちゃコミットできる
- → 結果が出やすい

**結果:**
- 絶対に5人に分散するより、2人に集中してサポートできた方が成果につながりやすい
- 結果が出るので、長期的に実績もたまる
- 長期的に稼ぎ続けられる

#### メリット2: 顧客の成果が出やすい

**高単価の場合:**

```
少人数 → リソース集中 → 手厚いサポート → 成果が出る → 実績がたまる → 長期的に稼げる
```

**低単価の場合:**

```
多人数 → リソース分散 → サポートが薄い → 成果が出ない → 実績がたまらない → 長期的に稼げない
```

**→ 低単価商品にメリットは1個もない**

### 低単価商品のダメなパターン

**絶対にやってはいけない:**

| ダメな商品 | 理由 |
|----------|------|
| 低単価のnoteを販売 | バックエンドにつながらない |
| 低単価のBrainを販売 | バックエンドにつながらない |
| サポートなしの商品 | 結果が出ない |
| オンラインサロンで低単価 | めちゃめちゃ悪手 |

**→ コンパス講座生は絶対にやらないでください**

### 低単価商品がバックエンドにつながらない理由

**3,000円のnoteを例に:**

**問題1: 商品の質が高すぎる場合**
```
満足されてしまう → バックエンド商品につながらない
```

**問題2: 商品の質が低すぎる場合**
```
満足されない → バックエンド商品につながらない
```

**結論:**

```
どっちに転んでも、低単価商品はバックエンドにつながらない
```

**理由:**
- 入れる情報をどのくらい入れていいかが難しすぎる
- 入れなさすぎても、どうしようもない商品になる
- 入れ過ぎても、良すぎる商品になって、高単価に変わらない

**不思議な真実:**
- 良すぎても高単価に変わらない

**→ だから、高単価で絶対に販売した方がいい**

### 高単価で販売するためには

#### 方法1: 購買欲求の高い需要を満たす

**3大ニッチ:**
- 健康市場
- お金の市場
- 恋愛の市場

**拡張版:**
- 健康、美容、お金、恋愛、将来

**→ 購買欲求が高い需要を満たせるように、スキル・実績をつける**

#### 方法2: ファン化してもらう

**ファン化の効果:**

```
あなたからどうしても学びたい
↓
ぜひお金払ってでも学ばせてください
↓
ノールックで高単価を買ってくれる
```

**→ そういうお客さんが結構集まってくる**

#### 方法3: ブランディング

**重要な考え方:**

```
高いイメージをつける = ブランディング
```

**ブランディングの例:**

| 発信内容 | イメージ |
|---------|---------|
| インフルエンサーと仕事をしている | こいつはヤバい、商品も高そう |
| 夜遅くまでめちゃめちゃ仕事してる | めちゃめちゃ頑張ってる、ファン化する |

**→ 日々の発信から高いイメージをつける**

## 7. 価格設定の3つの方法

### 全体像

**3つの方法:**

1. **業界の相場**
2. **投資対効果**
3. **モニター価格**

### 方法1: 業界の相場

#### 定義

**考え方:**

```
業界の相場・顧客のイメージ価格によって決める
```

**→ 手に入る結果に対して払う価格の相場**

#### 具体例: ライザップ

**業界ナンバーワンを基準にする:**

| サービス | 期間 | 価格 |
|---------|------|------|
| ライザップ | 2ヶ月 | 40万円 |
| 類似商品(トレントワークアウト) | 2ヶ月 | 25万円 |

**ポイント:**
- 最終的な価格は似ている
- 顧客のイメージ価格として存在している
- 業界の相場ができている

**→ この相場をもとに商品の価格を決める**

**その上で高単価にする**

### 方法2: 投資対効果

#### 定義

**考え方:**

```
顧客が将来いくら得するのか?
今払って、お金がどのくらい節約されるのか?
```

**→ そういった視点で価格設定する**

#### 投資対効果の表現方法

**見せるべきこと:**

```
現状で使っている時間・労力・お金
↓
自分のサービスを購入すると
↓
どれほど節約できるのか?
将来的にいくらぐらい得するのか?
```

**→ 未来を見せた上で価格を提示する**

#### 例1: 稼ぐ系(簡単)

**コンパスの場合:**

```
50万円の投資 → 6ヶ月後に100万円稼げる → 50万円が戻ってくる
```

**→ 未来が見せやすい、納得しやすい**

#### 例2: 美容市場(難しい)

**問題:**
- 美容で50万円です → 50万円高くない?と思われる

**解決策: 投資対効果を見せる**

**現状の分析:**

| 項目 | 現状 | 年間換算 |
|------|------|---------|
| 高級美容グッズ | 毎月5万円 | 60万円 |
| スキンケア時間 | 一日2時間(月60時間) | 720時間 |
| **合計コスト** | **— | 60万円 + 720時間** |

**自分のメソッドを使うと:**

| 項目 | 改善後 | 年間換算 |
|------|--------|---------|
| 美容グッズ | 毎月2万円 | 24万円 |
| スキンケア時間 | 一日30分(月15時間) | 180時間 |
| **合計コスト** | **— | 24万円 + 180時間** |

**差分:**

```
60万円 - 24万円 = 36万円の節約
720時間 - 180時間 = 540時間の節約
```

**価格設定:**

```
36万円と540時間の節約ができる
→ 30万円〜60万円で設定できる
→ 60万円でも納得してもらえる
```

**→ 投資対効果を見せると、高単価でも納得してもらえる**

#### 将来のリターンに対する対等な価値

**シンプルな例:**

```
500万円払って、2,000万円戻ってくる → みんな払う
```

**→ 将来のリターンに対する対等な価値を提供できるなら、この方法で価格設定できる**

### 方法3: モニター価格

#### モニター価格の効果

**メリット:**
- めちゃくちゃ売れやすい
- 成約率が上がる

#### モニター販売の意外な事実

**驚きの事実:**

```
大御所インフルエンサーでも、未だにモニター販売している人がいる
```

**実例:**
- 1年ぐらい同じ講座を売っている
- 講座生が400人くらい
- 未だにモニター販売

**理由:**
- モニター販売は、めちゃくちゃ売れやすい
- 成約率が上がる
- しかも単価を上げている

**→ めちゃめちゃ効果的**

#### いつまでモニター販売するか?

**推奨:**

```
次の桁に行くまでは、ずっとモニター販売でOK
```

**例:**
- 月収10万円まで → モニター販売
- 月収100万円まで → モニター販売
- 月収200万円まで → モニター販売

**→ ゼロイチは必ずモニター価格で**

#### モニター価格の心理

**お客さんの心理:**

```
モニター販売 = 安い
↓
安いのに、同じ商品を提供してもらえたらいいな
↓
成約率めちゃくちゃ上がる
```

### 価値に見合った商品を提供する

**忘れてはいけないこと:**

```
30万円の商品 → 30万円以上、お客様を満足させる
```

**→ めちゃめちゃ大事**

### モニター価格の設定方法

**計算方法:**

```
業界の相場または投資対効果で設定した価格
↓
50〜90%オフ
```

**推奨:**
- 80%オフ
- 50%オフ

**→ この範囲で調整**

### 高単価の方が結果が出る

**重要な原則:**

```
高単価のモニター販売 > 低単価のモニター販売
```

**理由:**
- 高単価の方が、お客さんが結果を出す
- 高単価の方が、本気で取り組む

**→ なるべく高単価の方がいい**

## 8. まとめ

### 商品設計の5ステップ(完全版)

#### ステップ1: 目次と提供内容の洗い出し
- 大まかな商品の全体像を設計
- 大目次・中目次・小目次を作る
- **ゼロイチの人はここまででOK**

#### ステップ2: コンテンツ内容の設計
- キュレーションで情報を収集
- ラフ設計 → 体系化 → 清書 → コンテンツ化
- スライド化・記事化・動画化
- テンプレート作成

#### ステップ3: ロードマップ・タスク化
- 顧客のカリキュラムを作る
- 各章ごとのタスクを明確にする
- マンツーマンコンサルでもタスク管理は必須

#### ステップ4: 提供スタイルの決定

**1. 保証の設計**

| 推奨 | 内容 |
|------|------|
| ○ アフターサポート保証 | 結果が出るまでサポート(絶対に入れる) |
| × 返金保証 | お客さんがやらなくなる(入れない) |

**2. 限定特典の設計**

```
特典を舐めるな! = メインに負けないクオリティで
```

| ポイント | 説明 |
|---------|------|
| メインの弱点を補う | あらかじめ突っ込まれそうなところを補う |
| メインに負けないクオリティ | 特典だけでも価値がある |
| 単体で商品を購入したいレベル | 30万円の商品なら、30万円分の価値 |

**3. 分割決済の設計**

| 商品価格 | 推奨分割回数 |
|---------|------------|
| 30万円未満 | 12分割 |
| 30万円以上 | 24分割 |

**4. サポート体制の設計**

| 必須サポート | 説明 |
|------------|------|
| チャットサポート | 当たり前 |
| 講義動画形式フィードバック(Loom) | 絶対に入れる |
| 個別添削 | 当たり前 |
| タスク管理 | 次に何をすればいいか明確にする |

**講座形式の場合:**
- 月1〜2回のZoomコンサル
- グループコンサル
- 録画配布

#### ステップ5: 価格設定

**鉄則:**

```
高単価商品一択
```

**絶対にNG:**
- 低単価商品
- オンラインサロン
- サポートなしの商品

**価格設定の3つの方法:**

| 方法 | 説明 |
|------|------|
| 1. 業界の相場 | 業界ナンバーワンを基準にする |
| 2. 投資対効果 | 節約できる金額・時間を見せる |
| 3. モニター価格 | 相場の50〜90%オフ |

**→ ゼロイチは必ずモニター価格で**

### 重要ポイント一覧

**商品設計の大前提:**
- ゼロイチはマンツーマンコンサルで、キュレーションでOK
- 商品コンセプト通りに設計する
- ゴールを必ず達成させることが最重要命題

**提供スタイルの決定:**
- アフターサポート保証は絶対に入れる(結果が出るまでサポート)
- 特典は豪華に!メインに負けないクオリティで
- 分割決済で成約率が圧倒的に向上
- サポート体制をめちゃめちゃ充実させる

**価格設定:**
- 高単価の方が、サポート人数が少なく、顧客の成果も出やすい
- 低単価商品にメリットは1個もない
- モニター価格でずっと売ってOK

**効率化:**
- Zoom録画を使えば動画コンテンツは効率的に作れる
- テンプレート・フレームワークでアクションに落とし込む
- Loomで動画形式フィードバック

### 高単価商品の方程式

```
高単価商品
↓
サポート人数が少ない
↓
リソース集中
↓
手厚いサポート
↓
成果が出る
↓
実績がたまる
↓
長期的に稼げる
```

**VS**

```
低単価商品
↓
サポート人数が多い
↓
リソース分散
↓
サポートが薄い
↓
成果が出ない
↓
実績がたまらない
↓
長期的に稼げない
```

### アクションプラン

**提供スタイルの決定:**
1. アフターサポート保証を設計する(結果が出るまでサポート)
2. 豪華な特典を3つ以上用意する
3. 分割決済を設定する(30万円以上なら24分割)
4. サポート体制を充実させる(Loom + チャット + 添削 + タスク管理)

**価格設定:**
1. 業界の相場を調べる(競合分析)
2. 投資対効果を計算する(節約できる金額・時間)
3. モニター価格を設定する(相場の50〜80%オフ)

**次のステップ:**
商品設計が完了したら、次はセールスとローンチの準備に進みます!
