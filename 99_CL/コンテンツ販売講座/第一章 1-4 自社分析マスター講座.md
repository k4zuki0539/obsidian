## 1. 自社分析で行うこと

### 自社分析とは?

**自分自身の経験・体験を深掘りしていく作業**

### なぜ自社分析が重要なのか?

SNSビジネスでは、**自分の経験・体験・価値観**を反映させることで、以下すべてに活用できる:

- ポジショニング
- キャラクター性
- コンセプト
- 商品設計

### SNSビジネスにおける自社分析の意義

**SNSビジネスの特徴:**
- 店舗やリアルビジネスよりも、個人の経験・体験・価値観が重要
- 自分の経験を反映させることで、オリジナリティが生まれる
- 独自のポジションを確立できる
- 大きな差別化が可能になる

**→ 自己分析で経験・体験を深掘りすることが成功の鍵**

## 2. ビフォーアフターの重要性

### ビフォーアフターとは?

**悩んでいた状態(Before) → 克服・解消した状態(After)**

- コンプレックスを乗り越えた経験
- ダメなところを解消できた過去
- 深刻な悩みからの脱却

### ビフォーアフターと収益の関係

**重要な法則:**

```
ビフォーアフターが明確 = 価値提供の度合いが大きい = 早く稼げる
```

#### 具体例

**肌荒れのビフォーアフター:**
- Before: 肌が荒れている状態
- After: 肌荒れを治すことができた
- → 肌荒れで悩む人に価値提供できる

### パターン別の戦略

#### パターン1: ビフォーアフターがある人

**現状:** すでに悩みを克服した経験がある

**強み:**
- すぐに価値提供できる
- 早く稼げるようになる

#### パターン2: ビフォーアフターがない人

**現状:** まだ悩みを克服した経験がない

**戦略:**
- 今から作ればOK
- 3〜6ヶ月の努力でビフォーアフターは作れる
- 繰り返すことで価値提供の度合いを増やしていく

**長期的視点:**
- 3〜6ヶ月のサイクルを繰り返す(1年、2年、3年...)
- ビフォーアフターの振れ幅が大きくなる
- 価値提供できる度合いが増大
- 長期的に稼げるようになる

### ビフォーアフターと価値観の関係

**ビフォーアフターが大きいほど、価値観の形成が強くなる**

影響を与える要素:
- 経験の長さ(期間が長いほど)
- 悩みの深刻さ(深刻なほど)
- 変化の振れ幅(大きいほど)

**結果:**
- 価値観の形成が強固になる
- 性格・キャラクターが明確になる
- SNSビジネスで強みになる

## 3. 自社分析で深掘る5つのこと

### 自社分析の5つのステップ

1. **どんな状況で**(過去の状況)
2. **何をしてきたか**(過程)
3. **どんな結果を得たのか**(結果)
4. **振り返ってどう思うか**(価値観)
5. **自分のアセット**(独自資源・能力)

### 各ステップの概要

#### 1. 過去の状況(Before)
- 自分が過去に置かれていた状況
- ビフォーの状態

#### 2. 過程(Process)
- ビフォーの状態で、具体的に何をしてきたのか
- どんな行動を取ったのか

#### 3. 結果(After)
- ビフォーで何をして、どんな結果を得たのか
- 達成した成果

#### 4. 価値観(Value)
- ビフォーアフターの過程で何を思ったのか
- どんな価値観が形成されたのか

**重要:** 人間は、現状から理想に向かって行動する中で、価値観が形成される

#### 5. アセット(Asset)
- **アセット = 独自資源・独自能力**
- 価値観をもとに自分のアセットを見極める
- 最終的なゴール

---

## 4. 各項目の解説

### 項目1: どんな状況で(過去の状況)

**書き出す内容:**
- 過去に置かれていた心境・状況
- 抱えていた深刻な悩み・コンプレックス
- 当時の自分の状態

**具体例:**
- うつ状態だった
- 自信が持てなかった
- 経済的に困窮していた
- 人間関係で悩んでいた

### 項目2: 何をしてきたか(過程)

**書き出す内容:**
- その悩んでいた状況で、自分が何をしたのか
- どんな行動を取ったのか
- どんな努力をしたのか

### 項目3: どんな結果を得たのか(結果)

**重要:** 定量的な結果と定性的な結果の**両方**を書き出す

#### 定量的な結果

**定量的結果 = 数値で表せるもの**

| ジャンル | 定量的な結果の例 |
|---------|----------------|
| ダイエット | 6ヶ月で15kg痩せた |
| 転職 | 3ヶ月で大手企業に転職、年収100万円アップ |
| 肌荒れ | 5ヶ月でニキビが無限→2個に減少 |

#### 定性的な結果

**定性的結果 = 感情的なベネフィット(数値では表せない)**

| ジャンル | 定性的な結果の例 |
|---------|----------------|
| ダイエット | 着たかった服が着れるようになり、買い物が楽しくなった |
| 転職 | 給料が安定し、週一の贅沢を我慢しなくて済み、精神的に安定した |
| 肌荒れ | 鏡を見るのが憂鬱だったが、きれいな肌を自慢したくなった |

### 項目4: 振り返ってどう思うか(価値観)

**価値観は経験・体験から必ず芽生える**

**書き出す内容:**

1. **業界・風習に対して**
   - その業界に対してどう思うか
   - 業界の風習についてどう感じるか

2. **自分自身に対して**
   - よく頑張ったと思うか
   - 行動の重要性を理解したか
   - どんなマインドが形成されたか

3. **過去の自分に対して**
   - 過去の状況にいた自分をどう思うか

### 項目5: 自分のアセット

**アセット = 自分の独自資源・能力**

**書き出す内容:**
- 自分だけの強み
- 得意なこと
- 価値提供できるスキル
- コンプレックスを乗り越えた経験
- 価値観・信念

**→ これは絶対に最終的に出すこと(最重要)**

## 5. 自社分析する期間

### 基本方針: 全期間を分析する

**推奨:** 人生全体をざっくりと分析する

### 例外: 大ジャンルに特化してもOK

**ケース例:**
- 現在41歳
- 大ジャンル: 恋愛
- 恋愛に取り組んだ期間: 31歳〜40歳
- 20歳〜30歳: 恋愛に関して特に何もしていない

**この場合:** 31歳〜40歳の期間だけ深掘りしてもOK

### できれば全部やった方がいい理由

**価値観の形成時期:**
- 価値観や性格の形成は25歳までに決まると言われている
- 幼少期に芽生えた価値観は特に重要
- 人生全体を振り返ることで、より深い自己理解が得られる

**→ 可能であれば、全期間を分析することを推奨**

## フレームワークの使い方

### フレームワークの入手方法

1. 概要欄にある「自社分析のフレームワーク」を開く
2. マインドマップをコピーして使用する
3. **必ずコピーして使うこと**

### フレームワークの構成

**年齢区分の例:**
- 15歳〜18歳
- 19歳〜22歳
- 23歳〜26歳
- 27歳〜30歳
- 40歳〜50歳

**→ 自分の年齢に合わせてカスタマイズする**

### 最重要項目: アセット

**講師が一番見ているポイント:**
- 自分のアセットがしっかり出ているか
- 全項目を書くことで、自己理解が深まる
- アセットを出すためには、全項目の記入が必要

**添削で分かること:**
- どんな経験をしてきたのか
- どんな能力があるのか
- どのジャンルが向いているか
- どんな発信の切り口が良いか

### 自社分析は誰のためにやるのか?

**一番の目的: 自分自身のため**

**自社分析をする理由:**
- SNSビジネスで稼いでいくため
- 価値観・性格・強み・弱みをビジネスに反映させるため

**自社分析で得られるもの:**
- キャラクターの独自性
- オリジナリティ
- パーソナリティ
- アイデンティティ

**→ これらが最終的にブランディングにつながる**

### 自己理解の重要性

**重要な原則:**
```
自分自身を理解していない = ビジネスに反映できない
自分の強み・弱みを知らない = 発信できない
```

**自社分析のゴール:**
- アセットをしっかり出す
- 自分で理解できていればOK

### 講師の例を参考にする

**参考資料:**
- マインドマップのリンクから講師の例が見られる
- 講師が何をしてきたのか
- 価値観の形成まで詳しく記載

**→ 講師の例を参考にしながらフレームワークに取り組む**

## まとめ

### 自社分析の重要性

**自社分析 = SNSビジネスで稼ぐための土台**

**活用できる領域:**
- ポジショニング
- キャラクター性
- コンセプト
- 商品設計

### ビフォーアフターの力

**重要な法則:**
```
ビフォーアフターが明確 = 価値提供の度合いが大きい = 早く稼げる
```

**ビフォーアフターの要素:**
- コンプレックスを乗り越えた経験
- ダメなところから解消できた過去
- 深刻な悩みからの脱却

### 自社分析で深掘る5つのこと

| 項目 | 内容 |
|------|------|
| 1. どんな状況で | 過去の状況、悩み、コンプレックス |
| 2. 何をしてきたか | 過程、行動、努力 |
| 3. どんな結果を得たのか | 定量的結果と定性的結果(両方必須) |
| 4. 振り返ってどう思うか | 価値観の形成、マインド |
| 5. 自分のアセット | **最重要**:強み、スキル、経験、価値観 |

### 分析期間

- **基本:** 人生全体を分析
- **例外:** 大ジャンルに関連する期間のみでもOK
- **推奨:** 25歳までの価値観形成期を含めると効果的

### 課題

**フレームワークを使って、自社分析をまとめて提出**

**最重要ポイント:**
- **アセット(自分の強み、価値観、スキル)をしっかり出すこと**
- 講師の例を参考にする
- 自分の人生を深掘りする

### 自社分析のゴール

```
自己理解 → ビジネスへの反映 → ブランディング → 収益化
```
