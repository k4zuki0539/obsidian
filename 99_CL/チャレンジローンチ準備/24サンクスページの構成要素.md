# サンクスページ（TP）の構成要素

## サンクスページとは

LP登録後に表示されるページ。  
オープンチャットへの誘導を完了させる重要なページ。

---

## 3つのパターン

| パターン | 特徴 | 使用場面 |
|---------|------|---------|
| ①動画+特典 | フル構成 | 熱量MAXで誘導 |
| ②動画のみ | 特典なし | シンプルに誘導 |
| ③動画・特典なし | 最小構成 | 素早く誘導 |

---

## ①動画+特典パターン（フル構成）

### 構成要素

```
1. 緊急アラートセクション
  ↓
2. 動画セクション（メリット・ベネフィット提示）
  ↓
3. 参加のために必要なアクションの提示
  ↓
4. CTA（1回目）
  ↓
5. 特典・ボーナスリスト
  ↓
6. オープンチャットの参加手順＆安全性の提示
  ↓
7. CTA（2回目）
  ↓
8. 特典リマインド
  ↓
9. フッター（会社情報）
```

---

### 1. 緊急アラートセクション

**目的：** 登録完了していないことを明示

| ポイント | 内容 |
|---------|------|
| 警告感 | 「まだ完了していない」と伝える |
| 次のアクション | 何をすべきか明示 |

**例：**
```
重要　まだ登録は完了していません
```

---

### 2. 動画セクション

**目的：** 熱量を高めて企画参加を促す

| 内容 |
|------|
| 企画のメリット・ベネフィットを伝える |
| 視聴者の熱量を高める |
| 企画への参加を促す |

---

### 3. 参加のために必要なアクションの提示

**目的：** 次のアクションを明確に

**例：**
```
LINEオープンチャットに参加しないと
（企画名）には参加できません
```

---

### 4. CTA（1回目）

#### ボタン上部
緊急性・希少性を伝える一文

**例：**
```
5月20日 20:59まで限定受付
```

#### ボタン内
企画の端的なメリット ＋ 行動喚起コピー

**例：**
```
今すぐ無料で最新AI活用が学べる
5日間の集中講座に参加する
```

---

### 5. 特典・ボーナスリスト

**目的：** 参加メリットを視覚的に訴求

| ポイント | 内容 |
|---------|------|
| 図で示す | 視覚的に表現 |
| 得られるもの | 参加によって得られる特典 |

---

### 6. オープンチャットの参加手順＆安全性の提示

#### 構成
| 要素 | 内容 |
|------|------|
| 小見出し | 「LINEオープンチャットとは？」 |
| 安心材料 | チェックリスト形式 |
| 手順解説 | STEP1〜STEP4 |
| キャプチャ | スマホ画面の具体的操作を解説 |

---

### 7. CTA（2回目）

→ 1回目のCTAと同様のコピーを配置

---

### 8. 特典リマインド

**目的：** ページ後半で再度訴求

| ポイント |
|---------|
| 特典画像をもう一度まとめて表示 |
| 離脱防止 |

---

### 9. フッター（会社情報）

| 項目 |
|------|
| 運営会社名 |
| 所在地 |
| 問い合わせ先リンク |

---

## CTAのコピー例

### ボタン上部（緊急性・希少性）
```
5月20日 20:59まで限定受付
```
```
残り◯名様限定
```
```
本日23:59締め切り
```

### ボタン内（メリット＋行動喚起）
```
今すぐ無料で最新AI活用が学べる5日間の集中講座に参加する
```
```
【無料】◯◯が身につく3daysセミナーに今すぐ参加する
```

---

## オープンチャット説明のポイント

### 安心材料チェックリスト例
- ✅ LINEの友だちに通知されない
- ✅ 匿名で参加できる
- ✅ いつでも退出可能
- ✅ 個人情報は公開されない

### STEP解説
| STEP | 内容 |
|------|------|
| STEP1 | ボタンをタップ |
| STEP2 | 「参加」をタップ |
| STEP3 | ニックネームを入力 |
| STEP4 | 参加完了 |

※スマホ画面キャプチャで具体的に解説

---

## まとめ

> ✅ サンクスページは**OC誘導完了**が目的  
> ✅ **緊急アラート**で「まだ完了していない」と伝える  
> ✅ CTAは**2回配置**（上部・下部）  
> ✅ **参加手順＋安全性**で不安を払拭  
> ✅ 3パターンから企画に合ったものを選択  
> ✅ フル構成なら特典リストで追加訴求
