# チャレンジローンチのリアル要素

## 概要

### リアル要素とは
- 課題のアウトプット
- ライブセミナー終了後にユーザーに何かしらアクションを起こしてもらうもの

---

## リアル要素の目的

### 1. 一体感の演出
- 雰囲気を良好にする
- 学びの質を高める

### 2. 成長の可視化
- 課題をこなす過程でライブ内容を振り返る
- 小さな成功体験の可視化
- 「こういうことができるようになった」という実感

### 3. 教育の深化
- 見てほしいコンテンツへの誘導
- 入れたい教育をさらに深く入れ込む

---

## リアルタイムコミュニケーションの効果

| 効果 | 内容 |
|------|------|
| 盛り上げコントロール | オープンチャットでのユーザーの動きをコントロール |
| 体験型価値の演出 | アウトプットへのアクションで価値を演出 |
| 熱量UP | セールスパート時のチャット盛り上げで成約率向上 |
| 喪失感の演出 | 全日程終了後の「終わり感」で次回オファーの反応率UP |

---

## 課題アウトプットの考え方

### 理想のアウトプット
> ライブの内容が**理想未来の実現に直結**することを体感させる

### 課題設計の例

**例1：プロンプト×キラーワードがテーマの場合**
- 課題1：今日の学びのアウトプット
- 課題2：どんな業務を効率化するプロンプトを作りたいか

**例2：GPTs作成がテーマの場合**
- 課題1：今日の学びのアウトプット
- 課題2：具体的にどんなGPTsを作りたいか

### 課題設計のポイント
| ポイント | 内容 |
|---------|------|
| デフォルト | 今日の学びのアウトプット・感想提出は必須 |
| 課題数 | 多くても**3つまで** |
| 重さ | 次回までに間に合うよう調整 |
| 内容 | 認識を強めてほしいところを課題に |

---

## 課題アウトプットの提出方法

### 1. オープンチャットのノート

| 項目 | 内容 |
|------|------|
| 手軽さ | ◎ ツールをまたがない |
| 用途 | ライトな課題向け |
| 制限 | 画像・リンクは送れない |
| 適した内容 | 感想、選択肢のアウトプット |

**おすすめ度：◎（一番ライト）**

---

### 2. オープンチャット直書き

| 項目 | 内容 |
|------|------|
| 手軽さ | ◎ 書き込むだけ |
| 一体感 | ◎ ユーザー間でアウトプット確認可能 |
| 注意点 | ネガティブコメントで雰囲気が引っ張られる |
| 通知 | 書き込み通知が増える → 離脱リスク |
| 推奨規模 | **100名以下** |

**メリット：**
- 盛り上がっているように見える
- 参加者の熱量が上がりやすい

**デメリット：**
- 100名超えると通知が多すぎる
- 運営でコントロールが必要

---

### 3. Googleフォーム

| 項目 | 内容 |
|------|------|
| 管理 | ◎ スプレッドシートに出力 |
| 突合 | ◎ 未提出管理・退出対応可能 |
| 設計 | 設問・入力情報をわかりやすく設定可能 |
| 工数 | 少ない |

**できること：**
- オープンチャット名・LINE名を取得
- 未提出者を検索して退出させる

**おすすめ度：◎（管理したい場合）**

---

### 4. 宴/エルメのフォーム

| 項目 | 内容 |
|------|------|
| 仕組み | アウトプット専用LINEを作成 |
| メリット | アクションしたホットリストをセグメント可能 |
| 活用 | 別セールス配信が可能 |
| 工数 | シナリオ用意が必要 |

**ツール選択：**
| ツール | 料金 |
|--------|------|
| Lステップ | 月額課金あり |
| **エルメ/宴** | 追加料金なし ← おすすめ |

**注意点：**
- 課題提出という認識で入っている
- 過度なセールス配信は離脱につながる

---

### 5. SNS（X/Instagram）

| 項目 | 内容 |
|------|------|
| 仕組み | 企画ポストを引用してアウトプット |
| メリット | フォロワーが見る → 参加者増加 |
| 条件 | **途中参加OKの企画限定** |
| 推奨規模 | **オープンチャット1000名以上** |

**メリット：**
- 企画が盛り上がっている印象を与えられる
- 途中参加者を増やせる
- 拡散効果

**デメリット：**
- 書き込みが少ないとネガティブ印象
- Xステップ/Iステップの設計工数がかかる
- 心理的ハードルが高い

**成功事例：**
- 500件以上のアウトプットが集まるような大型企画向け

## おすすめの選び方

### ライトなアウトプット
→ **オープンチャットのノート**

### 課題未提出者の管理・退出をしたい
→ **Googleフォーム**

### ホットリストを別でセグメントしたい
→ **宴/エルメのフォーム**

### 大型企画で拡散を狙いたい
→ **SNS（1000名以上推奨）**

---

## まとめ

> ✅ リアル要素 = 課題アウトプット + アクション促進  
> ✅ 目的：一体感、成長の可視化、教育の深化  
> ✅ 課題は**3つまで**、次回までに間に合う量で  
> ✅ おすすめ：**オープンチャットのノート** or **Googleフォーム**  
> ✅ 大型企画（1000名〜）はSNSも検討  
> ✅ 100名超えたらOC直書きは通知が多すぎる
