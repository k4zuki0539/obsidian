# 誘導動画の構成要素

## 全体の流れ

```
① オープニング導入
  ↓
② 共感・問題提起
  ↓
③ 権威付け・信頼構築
  ↓
④ オファー概要の提示
  ↓
⑤ セミナー/ライブの詳細
  ↓
⑥ 社会的証明
  ↓
⑦ 特典の訴求
  ↓
⑧ 締め切り・緊急性の再強調
  ↓
⑨ 行動喚起（CTA）
  ↓
⑩ 後押しのクロージング
```

---

## 1. オープニング導入

### 前提
- この時点でユーザーはあなたに**興味がないかもしれない**
- 「ちょっと覗いてみようかな」程度の気持ち
- いきなり自己紹介をしても興味がない

### やるべきこと
| アクション | 目的 |
|-----------|------|
| 得られるメリット・ベネフィットを提示 | 最後まで見てもらう |
| 視聴者を自分ごと化させる | 視聴を続ける理由を作る |
| 視聴を続ける理由を一文で提示 | 動画自体の期待値を高める |

---

## 2. 共感・問題提起

### やるべきこと
- 具体的な悩みを羅列
- ストーリーを通じて悩みに共感
- 自分ごと化させる

### 損失の見せ方

| NG | OK |
|----|-----|
| 最悪の未来を見せる | 悩みを放置した先の**イメージできる範囲の損失** |
| 不快な感情を呼び起こす | 機会損失を自認させる |

**ポイント：**
> 「この動画を見ないと損」「この企画に参加しないと損」という感情を起こす

### なぜ最悪の未来がNGか
- ユーザーの消費行動の変化
- 最悪の未来は刺さらなくなってきている
- 不快な感情は逆効果

---

## 3. 権威付け・信頼構築

### 目的
> 企画の内容をレクチャーする**資格がある**ことを認識させる

### ここで初めて自分の話をする

**入れるべき内容：**
| 要素 | 注意点 |
|------|--------|
| 実績 | ユーザーの課題解決につながるものに絞る |
| 経歴 | 端的に伝える |

**NG：** 関係ない実績の羅列

---

## 4. オファー概要の提示

### 伝えるべき内容

| 項目 | 内容 |
|------|------|
| 企画名 | コンセプトで作成済み |
| 開催形式 | ライブ/セミナー等 |
| 開催期間 | 日程 |
| 価格 | 無料 or 有料 |
| 参加メリット | **一文で約束** |

---

## 5. セミナー/ライブの詳細

### この時点でのユーザーの状態
- 企画内容に興味付けされている
- 講師に資格があることを認識している
- 「で、自分にはどんなメリットがあるの？」と聞いている

### 伝えるべき内容
| 項目 | 目的 |
|------|------|
| 企画のコンセプト | 全体像を伝える |
| 得られるメリット | 具体的に端的に |
| 到達するゴール | イメージを沸かせる |

**目的：** ここで一番**受講意欲を高める**

---

## 6. 社会的証明

### 目的
> 5番で提示したメリット・ベネフィット・ゴールが**本当に訪れる証拠**を並べる

### 入れるべき内容
| 要素 | 内容 |
|------|------|
| 受講生の事例 | 成果を出した人 |
| 類似の環境での成果 | ターゲットに近い事例 |
| 自分自身のストーリー | 手法の有効性 |

**ポイント：**
- 成果を**自分ごと**としてイメージしてもらう
- ターゲットユーザーに合わせて調整

---

## 7. 特典の訴求

### 目的
> **今アクションする理由**を作る

### 興味付け → 行動の後押しへ

```
4〜6番：企画自体の興味付け・期待値UP
7番〜：今アクションする理由を固める
```

### 特典の種類
- 参加特典
- 課題提出特典
- 全日程コンプリート特典

### 効果的な特典の特徴

| 特徴 | 説明 |
|------|------|
| **即効性** | すぐに使える |
| **体験型価値** | 参加することで得られる |
| **限定性** | 限定数・期間 |
| **希少性** | ここでしか手に入らない |

### 特典の例
| 反応が取れる | 反応が取れにくい |
|-------------|-----------------|
| ツール | とりあえず用意したコンテンツ |
| テンプレート | 価値の薄い情報 |
| 圧倒的ボリュームのコンテンツ | ありきたりな内容 |

**ポイント：**
> 理想未来の実現や企画で得られる成長を**強化する**特典に絞る

---

## 8. 締め切り・緊急性の再強調

### 伝えるべき内容
| 要素 | 内容 |
|------|------|
| タイムリミット | いつまでか |
| 今すぐの行動を促す | 後回しにさせない |
| 期限後の損失 | 参加できなくなる |
| 機会の希少性 | 今だけ |

---

## 9. 行動喚起（CTA）

### やるべきこと
- 具体的なアクションを示す
- すべきことを明確にする

### 伝え方の例
```
「ここから企画に参加できます」
「今こういうアクションを取ったら登録完了です」
```

**ポイント：** 明確なCTAを整理して提示

---

## 10. 後押しのクロージング

### やるべきこと
- 講座・企画のメリットを最後にもう一度伝える
- 行動を促して終了

---

## 構成要素のまとめ

| 番号 | 要素 | 目的 |
|------|------|------|
| 1 | オープニング導入 | 最後まで見てもらう |
| 2 | 共感・問題提起 | 自分ごと化 |
| 3 | 権威付け・信頼構築 | 資格を認識させる |
| 4 | オファー概要 | 企画の全体像 |
| 5 | セミナー詳細 | 受講意欲を高める |
| 6 | 社会的証明 | 成果のイメージ |
| 7 | 特典の訴求 | 今行動する理由 |
| 8 | 締め切り・緊急性 | 後回しにさせない |
| 9 | CTA | 具体的なアクション |
| 10 | クロージング | 最後の後押し |

---

## 活用のポイント

### 流れは前後してもOK
- 要素がすべて含まれていれば問題なし
- 自分のスタイルに合わせて調整

### AI活用
- この要素をAIに読み込ませる
- 背景情報・企画の目的も読み込ませる
- 動画台本を出力させる

---

## まとめ

> ✅ 10個の構成要素をすべて含める  
> ✅ 冒頭は自己紹介ではなく**メリット提示**  
> ✅ 最悪の未来ではなく**機会損失**を見せる  
> ✅ 4〜6番は**興味付け**、7番〜は**行動の後押し**  
> ✅ 特典は**即効性・限定性・希少性**を意識  
> ✅ CTAは**具体的に明確に**
