# LINE Obsidian システム設定

## システム概要
LINEメッセージをObsidian vaultに自動的に取り込み、整理するシステム

## フォルダ構造
```
vault/
├── LINE inbox/           # LINEメッセージの一時保存場所
│   ├── daily/           # 日別メッセージ
│   ├── contacts/        # 連絡先別メッセージ
│   └── media/           # 画像・動画・ファイル
├── processed/           # 処理済みメッセージ
└── templates/           # メッセージテンプレート
```

## 主な機能
1. LINEメッセージの自動取り込み
2. 日付・連絡先別の自動分類
3. メディアファイルの整理
4. タグ付けとリンク作成

## 設定項目
- 自動処理間隔: 30分
- バックアップ頻度: 毎日
- 保持期間: 6ヶ月

## テンプレート
- 日次メッセージテンプレート
- 連絡先テンプレート
- メディアファイルテンプレート

## 使用方法
1. LINEから取得したメッセージを「LINE inbox/daily」フォルダに配置
2. 適切なテンプレートを使用してノートを作成
3. 処理完了後は「processed」フォルダに移動
4. タグとリンクを使用して関連付けを作成

## 自動化のヒント
- Obsidian のテンプレータープラグインを使用
- 定期的にフォルダを整理
- 重要なメッセージには手動でタグ付け
